## 第14题：break和continue

**问题**：break和continue的区别及使用场景

**答案**：
```python
# break: 跳出整个循环
for i in range(10):
    if i == 5:
        break
    print(i)  # 0, 1, 2, 3, 4

# continue: 跳过当前迭代，继续下一次
for i in range(10):
    if i % 2 == 0:
        continue
    print(i)  # 1, 3, 5, 7, 9

# 嵌套循环中的break
for i in range(3):
    for j in range(3):
        if j == 1:
            break
        print(f"i={i}, j={j}")
```

## 第19题：global和nonlocal

**问题**：global和nonlocal关键字的作用

**答案**：
```python
# global: 声明全局变量
count = 0

def increment():
    global count
    count += 1

increment()
print(count)  # 1

# nonlocal: 声明外层函数的局部变量
def outer():
    x = 10

    def inner():
        nonlocal x
        x += 1
        print(f"inner: x = {x}")

    inner()
    print(f"outer: x = {x}")

outer()
# 输出:
# inner: x = 11
# outer: x = 11

# 不使用nonlocal的情况
def outer2():
    x = 10

    def inner():
        x = 20  # 创建新的局部变量
        print(f"inner: x = {x}")

    inner()
    print(f"outer: x = {x}")

outer2()
# 输出:
# inner: x = 20
# outer: x = 10
```

## 第26题：enumerate函数

**问题**：enumerate的作用和用法

**答案**：
```python
# 基本用法
fruits = ['apple', 'banana', 'orange']
for index, fruit in enumerate(fruits):
    print(f"{index}: {fruit}")
# 输出:
# 0: apple
# 1: banana
# 2: orange

# 指定起始索引
for index, fruit in enumerate(fruits, start=1):
    print(f"{index}: {fruit}")
# 输出:
# 1: apple
# 2: banana
# 3: orange

# 获取索引和值的元组列表
indexed_fruits = list(enumerate(fruits))
print(indexed_fruits)  # [(0, 'apple'), (1, 'banana'), (2, 'orange')]

# 实际应用
def find_index(lst, target):
    for index, value in enumerate(lst):
        if value == target:
            return index
    return -1

print(find_index([1, 2, 3, 4], 3))  # 2
```

## 第27题：zip函数

**问题**：zip函数的使用场景

**答案**：
```python
# 同时遍历多个列表
names = ['Alice', 'Bob', 'Charlie']
ages = [25, 30, 35]
cities = ['NYC', 'LA', 'Chicago']

for name, age, city in zip(names, ages, cities):
    print(f"{name}, {age}, {city}")

# 创建字典
keys = ['name', 'age', 'city']
values = ['Alice', 25, 'NYC']
d = dict(zip(keys, values))
print(d)  # {'name': 'Alice', 'age': 25, 'city': 'NYC'}

# 转置矩阵
matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]
transposed = list(zip(*matrix))
print(transposed)  # [(1, 4, 7), (2, 5, 8), (3, 6, 9)]

# zip长度不一致（取最短）
a = [1, 2, 3]
b = ['a', 'b']
print(list(zip(a, b)))  # [(1, 'a'), (2, 'b')]

# 解压
pairs = [(1, 'a'), (2, 'b'), (3, 'c')]
numbers, letters = zip(*pairs)
print(numbers)  # (1, 2, 3)
print(letters)  # ('a', 'b', 'c')
```

## 第28题：else子句

**答案**：

# for-else: 循环正常结束时执行
for i in range(5):
    print(i)

def find_item(lst, target):
    for item in lst:
        if item == target:
            print(f"找到了: {target}")
            break
    else:
        print(f"未找到: {target}")

find_item([1, 2, 3, 4], 3)  # 找到了: 3
find_item([1, 2, 3, 4], 5)  # 未找到: 5

## 第29题：assert断言

**问题**：assert语句的作用

**答案**：
```python
# 基本用法
x = 10
assert x > 0, "x必须大于0"
# 如果条件为False，抛出AssertionError

# 示例
def divide(a, b):
    assert b != 0, "除数不能为0"

print(divide(10, 2))  # 5.0
# print(divide(10, 0))  # AssertionError: 除数不能为0

# 多个条件
age = 18
assert age >= 18, "年龄必须大于等于18"
assert age < 120, "年龄必须小于120"

# 注意：可以通过 python -O 禁用所有断言

# 推荐用法：检查函数前置条件
def withdraw(balance, amount):
    assert amount > 0, "取款金额必须大于0"
    assert amount <= balance, "余额不足"
    return balance - amount
```

## 第30题：推导式预览

**问题**：简单的列表推导式

**答案**：
```python
# 基本列表推导式
squares = [x**2 for x in range(10)]
print(squares)  # [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

# 带条件过滤
evens = [x for x in range(10) if x % 2 == 0]
print(evens)  # [0, 2, 4, 6, 8]

# 等价的传统写法
evens = []
for x in range(10):
    if x % 2 == 0:
        evens.append(x)

# 字符串处理
words = ['hello', 'world', 'python']
upper_words = [word.upper() for word in words]
print(upper_words)  # ['HELLO', 'WORLD', 'PYTHON']

# 条件表达式
nums = [1, 2, 3, 4, 5]
result = ['even' if x % 2 == 0 else 'odd' for x in nums]
print(result)  # ['odd', 'even', 'odd', 'even', 'odd']

# 嵌套循环
matrix = [[i*j for j in range(1, 4)] for i in range(1, 4)]
print(matrix)  # [[1, 2, 3], [2, 4, 6], [3, 6, 9]]
```
