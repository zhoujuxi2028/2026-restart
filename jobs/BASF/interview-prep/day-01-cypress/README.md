## 重点词汇表 (Key Vocabulary)

| 英文术语 | 中文 | 示例句 |
|---------|------|--------|
| Flaky tests | 不稳定测试 | "We reduced flaky tests by implementing proper wait strategies" |
| Test retry mechanism | 测试重试机制 | "Cypress has built-in retry logic for assertions" |
| Headless execution | 无头执行 | "We run tests in headless mode in CI/CD pipelines" |
| Test runner | 测试运行器 | "The Cypress test runner provides real-time feedback" |
| Automatic waiting | 自动等待 | "Cypress has automatic waiting built into every command" |
| Assertion | 断言 | "We use implicit and explicit assertions in Cypress" |
| Stub/Mock | 存根/模拟 | "We mock API responses using cy.intercept()" |
| Fixture | 测试数据文件 | "Fixtures help us manage test data consistently" |
| Custom command | 自定义命令 | "We created custom commands for authentication flows" |
| Time travel debugging | 时间旅行调试 | "Cypress allows time travel debugging with snapshots" |
