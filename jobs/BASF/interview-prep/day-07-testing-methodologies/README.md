# Day 7: Testing Methodologies + ISTQB

## Focus
STLC, BDD, OOP in testing, ISTQB fundamentals, Test Strategy & Planning

**Time**: 3-4 hours

## Learning Objectives
1. Explain STLC six phases and activities
2. Write BDD scenarios in Gherkin (Given-When-Then)
3. Describe Page Object Model pattern
4. Apply ISTQB test design techniques (Equivalence Partitioning, Boundary Value Analysis)
5. Discuss risk-based test strategy
