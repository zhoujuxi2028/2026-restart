# Day 5: Cloud Testing Fundamentals

## Focus
Cloud-based test environments, AWS/Azure/GCP platforms, cloud testing strategies

**Time**: 3-4 hours

## Learning Objectives
1. Explain cloud service models (IaaS, PaaS, SaaS) for testing
2. Compare AWS, Azure, GCP for QA purposes
3. Describe cloud testing architectures
4. Discuss Infrastructure as Code (IaC) basics
5. Explain cost optimization strategies

## Study Plan

### 1. Core Concepts (90 min)
Read `01-cloud-testing-concepts.md`:
- Cloud computing fundamentals
- Service models (IaaS/PaaS/SaaS)
- AWS, Azure, GCP comparison
- Cloud testing strategies

### 2. Interview Questions (60 min)
Study `02-interview-questions.md`:
- 10 cloud testing questions
- Prepare STAR format answers
- Record yourself (2-3 min per answer)

### 3. Practical Examples (30 min)
Review `03-cloud-examples/`:
- AWS Lambda test function
- Terraform configuration
- Cloud monitoring setup

### 4. English Practice (30 min)
Use `04-english-templates.md`:
- Cloud testing vocabulary
- Prepare 1-2 STAR stories
- Record explanations

## Success Criteria
- [ ] Explain IaaS vs PaaS vs SaaS for testing (2 min)
