## 🎯 岗位目标 (Objectives)

在 Global Digital Factory ECO (G-ECO/MD) 团队中，作为敏捷 DevOps 团队的一员，负责确保现代 Web 应用程序的质量。团队构建和运营 BASF Coatings 与外部客户共同使用的解决方案，通过系统化测试和持续质量改进，交付可靠、高性能、以用户为中心的数字产品。

## 📌 主要职责 (Main Tasks)

### 质量保障
- 交付高质量的 Intranet 和 Internet 应用程序，提升 BASF Coatings 对内外部客户的服务
- 定义测试策略，设计自动化测试解决方案，确保产品达成功能性和非功能性质量目标
- 深入理解业务逻辑、工作流程和用户场景，确保全面的测试覆盖

### DevOps 协作
- 在敏捷、自组织的 DevOps 团队中工作，支持产品全生命周期（从构想到上线及持续运营）
- 与软件开发人员紧密合作，验证新功能、早期发现缺陷、支持自动化测试
- 与产品负责人、产品经理、设计师和开发人员协作，确保对验收标准和质量期望的共识

### 测试自动化
- 设计、构建和维护自动化测试套件（功能测试、回归测试、性能测试）
- 确保测试套件集成到 CI/CD 流水线
- 使用适当工具（**Playwright, Cypress, Selenium, Postman, JMeter**）测试现代 Web 应用、云解决方案和 REST APIs

### 文档与监控
- 参与监控、性能测试和质量相关的 DevOps 活动，确保生产环境的可靠运行
- 记录测试策略、测试用例、测试结果和质量洞察，支持测试框架的透明度和可维护性
