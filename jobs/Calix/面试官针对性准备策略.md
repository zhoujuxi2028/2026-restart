**Q4: 如何使用Wireshark分析TCP性能问题？**
```
实战分析步骤：

1. 捕获策略
tcpdump -i eth0 -w tcp_analysis.pcap host target.server.com

2. Wireshark分析要点

TCP流分析：
- 右键数据包 → Follow → TCP Stream
- 分析完整的TCP会话流程

关键指标查看：
- RTT分析: Statistics → TCP Stream Graphs → Round Trip Time
- 窗口大小: Statistics → TCP Stream Graphs → Window Scaling
- 吞吐量: Statistics → TCP Stream Graphs → Throughput

3. 常见性能问题识别

窗口过小问题：
过滤器: tcp.analysis.window_full
现象: 接收端窗口满，发送端等待
解决: 调整接收缓冲区大小

重传问题：
过滤器: tcp.analysis.retransmission
现象: 大量重传包
原因: 网络丢包或延迟过高

慢启动问题：
现象: TCP拥塞窗口增长缓慢
分析: 观察cwnd变化曲线
优化: 调整初始窗口大小

4. 性能优化建议
- 启用窗口缩放选项
- 优化MSS设置避免分片
- 调整TCP拥塞控制算法
```

class APIClient:
    def __init__(self, base_url, timeout=30):
        self.session = requests.Session()
        self.base_url = base_url
        self.timeout = timeout

    def request(self, method, endpoint, **kwargs):
        url = f"{self.base_url}{endpoint}"

        # 请求前钩子
        self._pre_request_hook(method, url, kwargs)

        response = self.session.request(
            method, url, timeout=self.timeout, **kwargs
        )

        # 请求后钩子
        self._post_request_hook(response)

        return response

    def _pre_request_hook(self, method, url, kwargs):
        logger.info(f"{method} {url}")

    def _post_request_hook(self, response):
        logger.info(f"Response: {response.status_code}")

class BasePage:
    def __init__(self, driver):
        self.driver = driver
        self.wait = WebDriverWait(driver, 10)

    def find_element(self, locator, timeout=None):
        """增强的元素查找"""
        wait = WebDriverWait(self.driver, timeout or 10)
        try:
            return wait.until(EC.presence_of_element_located(locator))
        except TimeoutException:
            self._take_screenshot_on_failure()
            raise

    def _take_screenshot_on_failure(self):
        """失败时自动截图"""
        timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
        filename = f"failure_{timestamp}.png"
        self.driver.save_screenshot(f"screenshots/{filename}")
        allure.attach.file(f"screenshots/{filename}",
                          attachment_type=allure.attachment_type.PNG)

class SmartWait:
    @staticmethod
    def wait_for_page_load(driver, timeout=30):
        """等待页面完全加载"""
        WebDriverWait(driver, timeout).until(
            lambda d: d.execute_script("return document.readyState") == "complete"
        )

    @staticmethod
    def wait_for_ajax(driver, timeout=30):
