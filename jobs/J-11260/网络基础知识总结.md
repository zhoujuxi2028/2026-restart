# 网络基础知识总结
> 针对J-11260网络测试工程师职位面试准备

## 1. 以太网Layer 2/3协议核心

### Layer 2 (数据链路层)
**核心协议**：
- **以太网帧结构**：MAC地址、VLAN标签、帧检验序列
- **交换机工作原理**：MAC地址学习、帧转发、生成树协议(STP)
- **VLAN技术**：802.1Q标签、Trunk/Access端口配置

**面试重点**：
```
Q: 交换机如何学习MAC地址？
A: 交换机检查进入帧的源MAC地址，将其与进入端口关联存储在MAC地址表中，
   转发时查表决定从哪个端口发出，未知单播则泛洪到所有端口。
```

### Layer 3 (网络层)
**核心协议**：
- **IP路由**：静态路由、动态路由协议(OSPF、BGP)
- **子网划分**：CIDR、子网掩码计算
- **ARP协议**：IP地址到MAC地址的映射

**实战案例**：
```bash
# 路由表查看
ip route show
# ARP表查看
arp -a
# 子网连通性测试
ping -c 4 192.168.1.1
```

## 2. 网络管理协议

### SNMP (Simple Network Management Protocol)
**核心概念**：
- **SNMP版本**：v1/v2c/v3，安全性递增
- **组件**：Manager、Agent、MIB(管理信息库)
- **操作类型**：GET、SET、TRAP、WALK

**测试要点**：
```bash
# SNMP查询示例
snmpwalk -v2c -c public 192.168.1.1 1.3.6.1.2.1.1.1.0
# 系统信息获取
snmpget -v2c -c public 192.168.1.1 sysDescr.0
```

### NETCONF协议
**特性**：
- 基于XML的网络配置协议
- 支持事务性操作、配置验证
- 替代传统CLI配置方式

**操作示例**：
```xml
<rpc message-id="1" xmlns="urn:ietf:params:xml:ns:netconf:base:1.0">
  <get-config>
    <source><running/></source>
  </get-config>
</rpc>
```

## 3. 电信接入技术

### GPON/XGS-PON技术
**架构**：OLT(光线路终端) - ODN(光分配网络) - ONU/ONT(光网络单元)

**关键测试点**：
- **OMCI协议**：ONU管理与控制接口
- **功率预算**：光功率损耗计算
- **带宽分配**：DBA(动态带宽分配)算法
- **业务配置**：VLAN、QoS、组播等

### 常见测试场景
```
1. ONU注册流程测试
2. 业务开通端到端测试
3. 故障倒换与恢复测试
4. 性能基准测试(RFC2544)
```

## 4. 南北向接口架构

### 南向接口 (Southbound)
- **定义**：控制器到网元设备的接口
- **协议**：NETCONF、SNMP、OpenFlow、CLI
- **功能**：设备配置、状态监控、告警收集

### 北向接口 (Northbound)
- **定义**：上层应用到控制器的接口
- **形式**：REST API、GraphQL、Web界面
- **功能**：业务开通、网络可视化、运维操作

**架构示例**：
```
OSS/BSS应用层
     ↕ 北向API
  网管控制器
     ↕ 南向协议
   网络设备层
```
