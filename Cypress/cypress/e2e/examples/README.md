# 📚 示例和练习

本文件夹包含各种示例代码和挑战练习，用于巩固学习。

## 📁 文件夹说明

### `challenges/` - 挑战练习
包含各种难度的挑战测试，用于检验和提升技能。

**特点**:
- 🎯 难度递增
- 🔥 实战场景模拟
- 💪 技能综合应用

**文件**:
- `day3-challenges.cy.js` - Day 3 挑战练习
  - 复杂表单操作
  - 动态内容处理
  - 错误处理技巧
  - 最佳实践应用

### `demos/` - 演示示例
包含各种概念和技巧的演示代码。

**用途**:
- 📖 学习新概念
- 🔍 理解工作原理
- 💡 获取灵感

---

## 🎯 如何使用

### 1. 学习流程
```
学习每日课程 → 完成对应练习 → 尝试挑战题 → 查看演示示例
```

### 2. 挑战练习建议
- 先独立完成，不查看答案
- 遇到困难时，查看对应 Day 的学习笔记
- 完成后对比最佳实践
- 尝试优化自己的代码

### 3. 演示示例使用
- 阅读代码注释
- 运行测试观察行为
- 尝试修改参数实验
- 应用到自己的测试中

---

## 📝 练习建议

### 初学者
1. 完成基础练习（Day 1-3）
2. 尝试简单挑战
3. 参考演示示例

### 进阶学习者
1. 直接挑战高难度题目
2. 尝试多种解决方案
3. 优化测试代码质量

### 高级用户
1. 创建自己的挑战
2. 贡献最佳实践示例
3. 分享学习心得

## 🌟 推荐练习网站

1. **TodoMVC** - https://todomvc.com/
   - 适合练习基本操作
   - 多种框架实现可选

2. **Sauce Demo** - https://www.saucedemo.com/
   - 完整的电商测试场景
   - 包含登录、购物车、结账流程

3. **The Internet** - https://the-internet.herokuapp.com/
   - 各种特殊场景测试
   - 包含上传、拖拽、弹窗等

4. **Cypress Example** - https://example.cypress.io/
   - Cypress 官方示例网站
   - 覆盖所有核心命令
