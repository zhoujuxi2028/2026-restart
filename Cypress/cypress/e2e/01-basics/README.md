## 📝 测试文件

### Day 2: 第一个测试
**文件**: `day2-first-test.cy.js`

**学习内容**:
- ✅ `cy.visit()` - 访问网页
- ✅ `cy.get()` - 选择元素
- ✅ `cy.contains()` - 通过文本查找
- ✅ `.click()` - 点击操作

- ✅ `.should()` - 基本断言

**关键概念**:
- `describe` 和 `it` 的使用
- Cypress 命令的链式调用
- Time Travel 调试

### Day 3: 基本命令深度掌握
**文件**: `day3-basic-commands.cy.js`

**学习内容**:
- 🎯 `cy.visit()` 的各种用法（绝对路径、相对路径、参数）
- 🎯 `cy.get()` 的多种选择器（ID、class、属性等）
- 🎯 `.click()` 的高级用法（位置、选项、强制点击）
- 🎯 `.should()` 的各种断言类型

**选择器优先级**:
1. ⭐⭐⭐⭐⭐ `[data-cy="..."]` - 测试专用属性
2. ⭐⭐⭐⭐ `#id` - ID 选择器
3. ⭐⭐⭐ `.class` - Class 选择器
4. ⭐⭐ `tag` - 标签选择器

**关键技巧**:
- 链式断言的使用
- 条件性元素处理
- 动态内容等待

## 🎓 第一阶段总结

### 已掌握的技能 ✅

1. **测试结构**
   - 使用 `describe()` 组织测试套件
   - 使用 `it()` 编写测试用例
   - 理解测试的基本结构

2. **核心命令**
   - 页面导航：`cy.visit()`
   - 元素选择：`cy.get()`, `cy.contains()`
   - 交互操作：`.click()`, `.type()`, `.clear()`
   - 断言验证：`.should()`

3. **调试技巧**
   - Time Travel 调试
   - 使用 `.debug()` 暂停
   - Chrome DevTools 集成

4. **最佳实践**
   - 选择器优先级策略
   - 链式命令的正确使用
   - 利用自动等待机制
