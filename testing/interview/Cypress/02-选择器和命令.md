## 1. Cypress 支持哪些选择器类型？有什么推荐实践？

**答案**:

Cypress 支持所有标准 CSS 选择器，并推荐使用 data 属性。

```javascript
// 1. CSS 选择器
cy.get('.btn-primary');              // class
cy.get('#username');                 // id
cy.get('button');                    // 标签
cy.get('input[type="email"]');       // 属性
cy.get('.form > input');             // 子元素
cy.get('.item:first-child');         // 伪类

// 2. data-* 属性（推荐）
cy.get('[data-cy="submit-button"]');
cy.get('[data-test="login-form"]');
cy.get('[data-testid="user-name"]');

// 3. 包含文本
cy.contains('Submit');
cy.contains('button', 'Submit');
cy.contains('.nav-item', 'Home');

// 4. jQuery 扩展
cy.get('input:visible');
cy.get('button:enabled');
cy.get('li:contains("Active")');

// 5. 复杂选择器
cy.get('table tr:nth-child(2) td:first');
cy.get('.list-item').filter(':visible').first();
cy.get('form').find('input[type="text"]');
```

**推荐实践对比**:

```javascript
// ❌ 不推荐 - 易变的选择器
cy.get('.css-1x2y3z');              // CSS-in-JS 类名
cy.get('div > div > button');       // 依赖结构
cy.contains('Submit');              // 文本可能翻译

// ✅ 推荐 - data-cy 属性
cy.get('[data-cy="submit-btn"]');
cy.get('[data-cy="email-input"]');
cy.get('[data-cy="error-message"]');

// HTML 标记
<button data-cy="submit-btn" class="btn-primary">
  Submit
</button>
```

**自定义选择器命令**:

```javascript
// cypress/support/commands.js
Cypress.Commands.add('getByCy', (selector) => {
  return cy.get(`[data-cy="${selector}"]`);
});

Cypress.Commands.add('getByTestId', (selector) => {
  return cy.get(`[data-testid="${selector}"]`);
});

// 使用
cy.getByCy('login-button').click();
cy.getByTestId('user-avatar').should('be.visible');
```

**关键点**:
- 优先使用 `data-cy` 或 `data-testid`
- 避免依赖 CSS 类名和结构
- 使用语义化、描述性的属性值
- 考虑国际化时不依赖文本内容

## 2. cy.get() 和 cy.contains() 有什么区别？

**答案**:

`cy.get()` 通过选择器查找元素，`cy.contains()` 通过文本内容查找。

```javascript
// cy.get() - 选择器匹配
cy.get('button');                    // 所有 button
cy.get('.submit-btn');               // class 匹配
cy.get('[data-cy="login"]');         // 属性匹配

// cy.contains() - 文本匹配
cy.contains('Submit');               // 任何包含 "Submit" 的元素
cy.contains('button', 'Submit');     // 指定选择器内的文本
cy.contains('.btn', 'Submit');       // 组合选择器和文本

// 区别示例
// HTML: <button>Submit Form</button>
cy.get('button');                    // ✅ 找到
cy.contains('Submit');               // ✅ 找到（部分匹配）
cy.contains('Submit Form');          // ✅ 找到（完全匹配）
cy.get('button', 'Submit');          // ❌ 语法错误

// 复杂场景
// HTML:
// <div class="card">
//   <h3>Product A</h3>
//   <button>Add to Cart</button>
// </div>
// <div class="card">
//   <h3>Product B</h3>
//   <button>Add to Cart</button>
// </div>

// 选择 Product A 的按钮
cy.contains('.card', 'Product A')
  .find('button')
  .click();

// 或者
cy.contains('Product A')
  .parent()
  .find('button')
  .click();

// 精确文本匹配
cy.contains('button', 'Submit').click();           // 包含匹配
cy.contains('button', /^Submit$/).click();         // 正则精确匹配

// 大小写敏感性
cy.contains('Submit');                             // 大小写敏感
cy.contains(/submit/i);                            // 正则不区分大小写
```

**使用场景**:

```javascript
describe('选择器对比', () => {
  it('cy.get() 适用场景', () => {
    // 明确的元素标识
    cy.get('[data-cy="username"]').type('user');
    cy.get('#password').type('pass123');

    // 结构化查询
    cy.get('table tbody tr').should('have.length', 10);
  });

  it('cy.contains() 适用场景', () => {
    // 不确定具体选择器
    cy.contains('Login').click();

    // 动态内容验证
    cy.contains('Welcome back, John');

    // 多个相似元素中定位
    cy.contains('.notification', 'Success').should('be.visible');
  });
});
```

**关键点**:
- `cy.get()`: 通过 DOM 选择器查找
- `cy.contains()`: 通过文本内容查找
- `cy.contains()` 返回第一个匹配元素
- `cy.contains()` 可指定选择器范围
- `cy.contains()` 支持正则表达式

## 3. 如何使用 .find()、.children()、.parent() 等遍历命令？

**答案**:

Cypress 提供丰富的 DOM 遍历命令来导航元素关系。

```javascript
// 1. .find() - 查找后代元素
cy.get('.form').find('input');
cy.get('[data-cy="user-card"]').find('.avatar');

// 2. .children() - 查找直接子元素
cy.get('.list').children();
cy.get('.nav').children('.active');

// 3. .parent() - 查找父元素
cy.get('.error-message').parent();
cy.contains('Delete').parent('button').click();

// 4. .parents() - 查找所有祖先元素
cy.get('input').parents('form');
cy.get('.item').parents('.container');

// 5. .closest() - 查找最近的祖先元素
cy.get('input').closest('form');
cy.get('.icon').closest('.card');

// 6. .siblings() - 查找兄弟元素
cy.get('.active').siblings();
cy.get('.tab-1').siblings('.tab-active');

// 7. .next() / .prev() - 下一个/上一个兄弟
cy.get('.item-2').next();
cy.get('.item-3').prev();

// 8. .first() / .last() - 第一个/最后一个
cy.get('li').first();
cy.get('.menu-item').last().click();

// 9. .eq() - 指定索引
cy.get('tr').eq(2);          // 第 3 行（从 0 开始）
cy.get('button').eq(-1);     // 最后一个

// 10. .filter() - 过滤元素
cy.get('li').filter('.active');
cy.get('input').filter('[type="text"]');
```

**实际应用示例**:

```javascript
// 示例 1: 表格操作
cy.get('table')
  .find('tbody tr')
  .eq(2)
  .within(() => {
    cy.get('td').eq(0).should('contain', 'John');
    cy.get('td').eq(1).should('contain', 'Doe');
    cy.contains('button', 'Edit').click();
  });

// 示例 2: 卡片组件
cy.contains('.card', 'Product A')
  .within(() => {
    cy.get('.price').should('contain', '$19.99');
    cy.get('.stock').should('contain', 'In Stock');
    cy.get('button').contains('Add to Cart').click();
  });

// 示例 3: 表单验证
cy.get('input[name="email"]')
  .type('invalid-email')
  .blur()
  .closest('.form-group')
  .find('.error-message')
  .should('be.visible')
  .and('contain', 'Invalid email');

// 示例 4: 导航菜单
cy.get('.nav-menu')
  .children('.nav-item')
  .should('have.length', 5)
  .filter('.active')
  .should('have.length', 1);

// 示例 5: 复杂遍历
cy.get('[data-cy="user-list"]')
  .find('.user-item')
  .filter(':contains("Admin")')
  .first()
  .within(() => {
    cy.get('.edit-btn').click();
  });
```

**链式遍历**:

```javascript
// 组合多个遍历命令
cy.get('form')
  .find('.input-group')
  .first()
  .find('input')
  .should('have.attr', 'required');

// 向上再向下导航
cy.contains('Product A')
  .parents('.product-card')
  .find('.add-to-cart')
  .click();

// 兄弟元素导航
cy.get('.tab-1')
  .click()
  .next('.tab-2')
  .should('not.have.class', 'active')
  .next('.tab-3')
  .should('exist');
```

**关键点**:
- `.find()` 查找所有后代，`.children()` 只查找直接子元素
- `.closest()` 查找最近匹配的祖先
- `.within()` 限制后续命令的作用域
- 遍历命令可以链式调用
- 使用 `.eq()` 访问特定索引元素

## 4. .click() 命令有哪些选项和变体？

**答案**:

`.click()` 命令支持多种选项来模拟不同的点击行为。

```javascript
// 1. 基本点击
cy.get('button').click();

// 2. 位置点击
cy.get('.canvas').click('topLeft');
cy.get('.element').click('center');
cy.get('.box').click('bottomRight');

// 可用位置: topLeft, top, topRight, left, center, right,
//          bottomLeft, bottom, bottomRight

// 3. 坐标点击
cy.get('.canvas').click(100, 200);      // x=100, y=200
cy.get('.element').click(50, 50);

// 4. 强制点击（忽略可见性检查）
cy.get('.hidden-btn').click({ force: true });

// 5. 多次点击
cy.get('button').click({ multiple: true });  // 点击所有匹配元素

// 6. 修饰键点击
cy.get('a').click({ metaKey: true });    // Cmd/Win + Click
cy.get('a').click({ ctrlKey: true });    // Ctrl + Click
cy.get('a').click({ shiftKey: true });   // Shift + Click
cy.get('a').click({ altKey: true });     // Alt + Click

// 7. 组合修饰键
cy.get('.item').click({
  ctrlKey: true,
  shiftKey: true
});

// 8. 超时设置
cy.get('button').click({ timeout: 10000 });

// 9. 双击
cy.get('.file').dblclick();

// 10. 右键点击
cy.get('.context-menu-trigger').rightclick();
```

**实际应用场景**:

```javascript
describe('点击命令应用', () => {
  it('画布点击', () => {
    cy.get('#canvas')
      .click(150, 200)
      .click(300, 400);
  });

  it('链接在新标签打开', () => {
    // 模拟 Cmd/Ctrl + Click
    cy.get('a[href="/page"]')
      .click({ metaKey: true });
  });

  it('多选操作', () => {
    cy.get('.item-1').click();
    cy.get('.item-2').click({ ctrlKey: true });
    cy.get('.item-3').click({ ctrlKey: true });
  });

  it('隐藏元素点击', () => {
    // 某些动画或 CSS 可能使元素"不可点击"
    cy.get('.overlay-button').click({ force: true });
  });

  it('上下文菜单', () => {
    cy.get('.file').rightclick();
    cy.contains('Delete').click();
  });

  it('点击所有复选框', () => {
    cy.get('input[type="checkbox"]')
      .click({ multiple: true });
  });
});

// 等待元素可点击
cy.get('button')
  .should('be.enabled')
  .and('be.visible')
  .click();

// 点击前滚动到元素
cy.get('.footer-link')
  .scrollIntoView()
  .should('be.visible')
  .click();
```

**点击验证**:

```javascript
// Cypress 自动验证
cy.get('button').click();
// 自动检查:
// 1. 元素存在于 DOM
// 2. 元素可见
// 3. 元素未被覆盖
// 4. 元素未禁用
// 5. 元素未在动画中

// 绕过验证
cy.get('button').click({
  force: true,           // 跳过可操作性检查
  waitForAnimations: false  // 不等待动画完成
});
```

**关键点**:
- 支持位置参数和坐标参数
- `force: true` 绕过可见性检查
- 支持键盘修饰键组合
- 自动等待元素可点击
- `dblclick()` 和 `rightclick()` 用于特殊点击

## 5. .type() 命令如何使用？支持哪些特殊键？

**答案**:

`.type()` 命令模拟键盘输入，支持特殊键和修饰键。

```javascript
// 1. 基本输入
cy.get('input').type('Hello World');

// 2. 特殊键
cy.get('input').type('{enter}');
cy.get('input').type('{esc}');
cy.get('input').type('{backspace}');
cy.get('input').type('{del}');
cy.get('input').type('{selectall}');

// 3. 组合输入
cy.get('input')
  .type('Hello')
  .type('{selectall}')
  .type('World');

// 4. 修饰键
cy.get('input').type('{ctrl}a');          // Ctrl+A
cy.get('input').type('{cmd}c');           // Cmd+C (Mac) / Ctrl+C
cy.get('input').type('{shift}{enter}');   // Shift+Enter

// 5. 导航键
cy.get('input').type('{leftarrow}');
cy.get('input').type('{rightarrow}');
cy.get('input').type('{uparrow}');
cy.get('input').type('{downarrow}');
cy.get('input').type('{home}');
cy.get('input').type('{end}');
cy.get('input').type('{pageup}');
cy.get('input').type('{pagedown}');

// 6. 重复字符
cy.get('input').type('{backspace}{backspace}{backspace}');
cy.get('input').type('{del}'.repeat(5));

// 7. 选项配置
cy.get('input').type('text', {
  delay: 100,              // 每个字符延迟 100ms
  force: true,             // 强制输入
  parseSpecialCharSequences: false  // 将 {} 作为普通字符
});

// 8. 清除后输入
cy.get('input').clear().type('new text');
```

**特殊键列表**:

```javascript
// 常用特殊键
const specialKeys = {
  // 功能键
  '{enter}': '回车',
  '{esc}': 'Escape',
  '{backspace}': '退格',
  '{del}': 'Delete',
  '{selectall}': '全选',
  '{alt}': 'Alt',
  '{ctrl}': 'Control',
  '{cmd}': 'Command (Mac)',
  '{shift}': 'Shift',

  // 导航键
  '{leftarrow}': '←',
  '{rightarrow}': '→',
  '{uparrow}': '↑',
  '{downarrow}': '↓',
  '{home}': 'Home',
  '{end}': 'End',
  '{pageup}': 'Page Up',
  '{pagedown}': 'Page Down',
  '{insert}': 'Insert',

  // 其他
  '{tab}': 'Tab',
  '{{}}': '输入字面量 {}'
};
```

**实际应用示例**:

```javascript
describe('输入测试', () => {
  it('表单填写', () => {
    cy.get('[data-cy="first-name"]').type('John');
    cy.get('[data-cy="last-name"]').type('Doe');
    cy.get('[data-cy="email"]').type('john@example.com');
    cy.get('[data-cy="submit"]').click();
  });

  it('搜索功能', () => {
    cy.get('.search-input')
      .type('Cypress')
      .type('{enter}');
    cy.url().should('include', '/search?q=Cypress');
  });

  it('文本编辑', () => {
    cy.get('textarea')
      .type('First line{enter}')
      .type('Second line{enter}')
      .type('Third line');
  });

  it('快捷键操作', () => {
    cy.get('textarea')
      .type('Some text to copy')
      .type('{selectall}')
      .type('{ctrl}c')
      .clear()
      .type('{ctrl}v');
  });

  it('自动完成选择', () => {
    cy.get('.autocomplete')
      .type('Cyp')
      .type('{downarrow}')
      .type('{downarrow}')
      .type('{enter}');
  });

  it('输入特殊字符', () => {
    // 输入字面量 {}
    cy.get('input').type('Use {{} and {}}', {
      parseSpecialCharSequences: false
    });
  });

  it('慢速输入模拟', () => {
    // 模拟用户真实打字速度
    cy.get('input').type('Realistic typing', {
      delay: 100
    });
  });
});

// 密码输入
cy.get('[type="password"]')
  .type('MySecurePassword123!', { log: false });  // 不记录到日志

// 数字输入
cy.get('[type="number"]')
  .clear()
  .type('42');

// 日期输入
cy.get('[type="date"]')
  .type('2026-12-31');
```

**关键点**:
- 特殊键用 `{}` 包裹
- 支持修饰键组合
- `delay` 选项控制输入速度
- `log: false` 隐藏敏感信息
- `clear()` 先清除再输入
- `parseSpecialCharSequences: false` 输入字面量 `{}`

## 6. .should() 和 .and() 断言命令有什么区别？

**答案**:

`.should()` 和 `.and()` 功能相同，`.and()` 是 `.should()` 的别名，用于提高可读性。

```javascript
// 1. 功能完全相同
cy.get('button').should('be.visible');
cy.get('button').and('be.visible');

// 2. 链式断言 - 提高可读性
cy.get('input')
  .should('be.visible')
  .and('have.attr', 'placeholder', 'Enter email')
  .and('have.value', '')
  .and('be.enabled');

// 等同于
cy.get('input')
  .should('be.visible')
  .should('have.attr', 'placeholder', 'Enter email')
  .should('have.value', '')
  .should('be.enabled');

// 3. 语义化使用
cy.get('.alert')
  .should('be.visible')
  .and('contain', 'Success')
  .and('have.class', 'alert-success');

// 4. 多个断言
cy.get('.username')
  .should('exist')
  .and('be.visible')
  .and('not.be.disabled')
  .and('have.text', 'John Doe');

// 5. 使用函数断言
cy.get('div').should(($div) => {
  expect($div).to.have.length(1);
  expect($div.text()).to.include('Expected text');
});

// 链式函数断言
cy.get('div')
  .should(($div) => {
    expect($div).to.be.visible;
  })
  .and(($div) => {
    expect($div.text()).to.match(/\d+/);
  });
```

**常用断言示例**:

```javascript
describe('断言类型', () => {
  it('存在性断言', () => {
    cy.get('.element').should('exist');
    cy.get('.element').should('not.exist');
  });

  it('可见性断言', () => {
    cy.get('.element')
      .should('be.visible')
      .and('not.be.hidden');
  });

  it('状态断言', () => {
    cy.get('button')
      .should('be.enabled')
      .and('not.be.disabled');

    cy.get('input[type="checkbox"]')
      .should('be.checked')
      .and('be.selected');
  });

  it('属性断言', () => {
    cy.get('input')
      .should('have.attr', 'type', 'email')
      .and('have.attr', 'required')
      .and('have.class', 'form-control')
      .and('have.id', 'email-input');
  });

  it('文本断言', () => {
    cy.get('.message')
      .should('have.text', 'Hello')
      .and('contain', 'Hell')
      .and('not.contain', 'World');
  });

  it('值断言', () => {
    cy.get('input')
      .should('have.value', 'John')
      .and('have.length', 1);
  });

  it('CSS 断言', () => {
    cy.get('.element')
      .should('have.css', 'color', 'rgb(255, 0, 0)')
      .and('have.css', 'display', 'block');
  });

  it('数量断言', () => {
    cy.get('li')
      .should('have.length', 5)
      .and('have.length.at.least', 3)
      .and('have.length.below', 10);
  });
});
```

**复杂断言**:

```javascript
// 1. URL 断言
cy.url()
  .should('include', '/dashboard')
  .and('contain', 'userId=123')
  .and('not.include', 'error');

// 2. 窗口断言
cy.window()
  .should('have.property', 'localStorage')
  .and('have.property', 'sessionStorage');

// 3. 响应断言
cy.request('/api/users')
  .should((response) => {
    expect(response.status).to.eq(200);
    expect(response.body).to.have.length(10);
  })
  .and((response) => {
    expect(response.headers).to.have.property('content-type');
  });

// 4. 元素集合断言
cy.get('.item')
  .should('have.length', 3)
  .and(($items) => {
    expect($items.eq(0)).to.contain('First');
    expect($items.eq(1)).to.contain('Second');
    expect($items.eq(2)).to.contain('Third');
  });
```

**关键点**:
- `.and()` 是 `.should()` 的别名
- 用 `.and()` 链接多个断言更易读
- 支持字符串断言和函数断言
- 自动重试直到断言通过或超时
- 可在一个 `.should()` 中写多个 `expect`

## 7. 如何使用 .invoke() 调用元素方法？

**答案**:

`.invoke()` 用于调用 jQuery 元素的方法或访问属性。

```javascript
// 1. 调用 jQuery 方法
cy.get('input').invoke('val', 'new value');
cy.get('.element').invoke('text', 'New Text');
cy.get('.item').invoke('show');
cy.get('.item').invoke('hide');

// 2. 获取属性值
cy.get('input').invoke('val').should('eq', 'expected');
cy.get('.element').invoke('text').should('contain', 'Hello');
cy.get('a').invoke('attr', 'href').should('include', '/page');

// 3. 调用原生 DOM 方法
cy.get('input')
  .invoke('get', 0)  // 获取原生 DOM 元素
  .then(($el) => {
    $el.scrollIntoView();
  });

// 4. 链式调用
cy.get('input')
  .invoke('val')
  .then((value) => {
    cy.log('Input value:', value);
  });

// 5. 带参数的方法调用
cy.get('.element')
  .invoke('css', 'color')
  .should('eq', 'rgb(255, 0, 0)');

cy.get('select')
```

## 8. .its() 和 .invoke() 有什么区别？

**答案**:

`.its()` 用于获取属性，`.invoke()` 用于调用方法。

```javascript
// .its() - 获取属性
cy.get('button')
  .its('length')
  .should('eq', 1);

cy.window()
  .its('localStorage')
  .invoke('getItem', 'token');

// .invoke() - 调用方法
cy.get('input')
  .invoke('val')  // 调用 val() 方法
  .should('eq', 'text');

cy.get('button')
  .invoke('attr', 'disabled');  // 调用 attr() 方法
```

**详细对比**:

```javascript
describe('its vs invoke', () => {
  it('.its() 获取属性', () => {
    // 获取对象属性
    cy.get('input')
      .its('length')
      .should('eq', 1);

    // 获取嵌套属性
    cy.window()
      .its('document.title')
      .should('include', 'My App');

    cy.request('/api/user')
      .its('body.name')
      .should('eq', 'John');

    // 获取数组元素
    cy.get('li')
      .its('0')  // 第一个元素
      .should('contain', 'First');
  });

  it('.invoke() 调用方法', () => {
    // 调用 jQuery 方法
    cy.get('input')
      .invoke('val', 'new value');

    // 调用原生方法
    cy.get('.element')
      .invoke('text')
      .should('contain', 'Expected');

    // 带参数调用
    cy.get('a')
      .invoke('attr', 'href')
      .should('include', '/page');
  });
});
```

**使用场景对比**:

```javascript
// 场景 1: 访问对象属性 vs 调用获取方法
cy.get('form')
  .its('0')                    // .its() - 获取第一个元素（属性访问）
  .then(($form) => {
    // 使用原生 DOM 元素
  });

cy.get('form')
  .invoke('get', 0)            // .invoke() - 调用 get(0) 方法
  .then(($form) => {
    // 同样效果
  });

// 场景 2: 获取响应数据
cy.request('/api/data')
  .its('body.users')           // .its() - 访问嵌套属性
  .should('have.length', 10);

cy.request('/api/data')
  .then((response) => response.body.users)  // 手动访问
  .should('have.length', 10);

// 场景 3: Window 对象操作
cy.window()
  .its('innerWidth')           // .its() - 获取属性
  .should('be.gt', 1000);

cy.window()
  .invoke('alert', 'Hello');   // .invoke() - 调用方法

// 场景 4: 本地存储
cy.window()
  .its('localStorage')
  .invoke('getItem', 'token')  // 先获取对象，再调用方法
  .should('exist');

// 等同于
cy.window().then((win) => {
  const token = win.localStorage.getItem('token');
  expect(token).to.exist;
});
```

**嵌套调用**:

```javascript
// .its() 支持点语法访问嵌套属性
cy.request('/api/user')
  .its('body.profile.address.city')
  .should('eq', 'New York');

// 等同于
cy.request('/api/user')
  .its('body')
  .its('profile')
  .its('address')
  .its('city')
  .should('eq', 'New York');

// .invoke() 链式调用方法
cy.get('ul li')
  .invoke('first')             // 调用 first() 方法
  .invoke('text')              // 调用 text() 方法
  .should('eq', 'First Item');
```

**关键点对比**:

| 特性 | .its() | .invoke() |
|-----|--------|-----------|
| 用途 | 获取属性 | 调用方法 |
| 参数 | 属性名（字符串） | 方法名 + 参数 |
| 嵌套 | 支持点语法 | 需要链式调用 |
| 返回 | 属性值 | 方法返回值 |

**关键点**:
- `.its()` 用于属性访问，`.invoke()` 用于方法调用
- `.its()` 支持嵌套属性点语法
- `.invoke()` 可传递参数给方法
- 两者常配合使用
- 都支持链式调用

## 9. .within() 命令的作用和使用场景？

**答案**:

`.within()` 限制后续命令的作用域在指定元素内。

```javascript
  cy.get('button[type="submit"]').click();
});

// 不使用 .within() 的对比
cy.get('form input[name="email"]').type('user@example.com');
cy.get('form input[name="password"]').type('password123');

```

## 10. 如何使用 .each()、.spread()、.then() 处理元素集合？

**答案**:

这些命令用于迭代和处理多个元素或数组。

```javascript
// 1. .each() - 迭代元素集合
cy.get('li').each(($el, index, $list) => {
  cy.wrap($el).should('contain', 'Item');
  cy.log(`Item ${index}: ${$el.text()}`);
});

// 2. .spread() - 展开数组参数
cy.get('li').then(($lis) => {
  return Cypress._.map($lis, 'innerText');
}).spread((first, second, third) => {
  expect(first).to.contain('First');
  expect(second).to.contain('Second');
  expect(third).to.contain('Third');
});

// 3. .then() - 处理前一个命令的结果
cy.get('button').then(($btn) => {
  const text = $btn.text();
  cy.log('Button text:', text);
});
```

**详细应用**:

```javascript
describe('集合处理', () => {
  it('.each() 遍历元素', () => {
    cy.get('.product-card').each(($card, index) => {
      // 每个卡片验证
      cy.wrap($card).within(() => {
        cy.get('.title').should('exist');
        cy.get('.price').should('match', /\$\d+/);
      });

      // 特定索引处理
      if (index === 0) {
        cy.wrap($card).should('have.class', 'featured');
      }
    });
  });

  it('.each() 收集数据', () => {
    const prices = [];

    cy.get('.product .price').each(($price) => {
      prices.push(parseFloat($price.text().replace('$', '')));
    }).then(() => {
      // 所有价格收集完成
      const total = prices.reduce((sum, price) => sum + price, 0);
      cy.log('Total:', total);
      expect(total).to.be.gt(100);
    });
  });

  it('.then() 链式处理', () => {
    cy.get('input')
      .type('test')
      .then(($input) => {
        // 获取 jQuery 对象
        return $input.val();
      })
      .then((value) => {
        // 处理返回值
        expect(value).to.eq('test');
        return value.toUpperCase();
      })
      .then((upper) => {
        cy.log(upper);  // 'TEST'
      });
  });

  it('组合使用', () => {
    cy.get('table tbody tr').each(($row, index) => {
      cy.wrap($row).within(() => {
        cy.get('td').first().then(($td) => {
          const text = $td.text();
          cy.log(`Row ${index}: ${text}`);
        });
      });
    });
  });
});
```

## 11. .wrap() 命令的作用和使用场景？

**答案**:

`.wrap()` 将对象或值包装成 Cypress 可链式调用的对象。

```javascript
// 1. 包装 DOM 元素
cy.get('button').then(($btn) => {
  // $btn 是 jQuery 对象
  cy.wrap($btn).click();
});

// 2. 包装普通值
cy.wrap('hello').should('eq', 'hello');
cy.wrap(42).should('be.gt', 40);
cy.wrap([1, 2, 3]).should('have.length', 3);

// 3. 包装对象
const user = { name: 'John', age: 30 };
cy.wrap(user)
  .its('name')
  .should('eq', 'John');

// 4. 包装 Promise
cy.wrap(Promise.resolve('value'))
  .should('eq', 'value');

// 5. 包装函数返回值
const getValue = () => 'test';
cy.wrap(getValue()).should('eq', 'test');
```

**实际应用场景**:

```javascript

      cy.wrap($li)
        .should('be.visible')
        .and('contain', 'Item');
    });
  });

  it('包装外部数据', () => {
    const testData = {
      email: 'user@example.com',
      password: 'pass123'
    };

      cy.get('#email').type(data.email);
      cy.get('#password').type(data.password);
    });
  });

  it('与别名结合', () => {
    cy.wrap({ token: 'abc123' }).as('auth');

    cy.get('@auth')
      .its('token')
      .should('eq', 'abc123');
  });

  it('包装异步操作', () => {
    const fetchData = () => {
      return new Promise((resolve) => {
        setTimeout(() => resolve({ status: 'ok' }), 1000);
      });
    };

    cy.wrap(fetchData())
      .its('status')
      .should('eq', 'ok');
  });

      .invoke('text')

  });
});
```

```javascript
    return cy.wrap('Normal handling');
  }
}).then((message) => {
  cy.log(message);
});
```

**关键点**:
- 将非 Cypress 对象转为可链式调用对象
- 常用于 `.then()` 回调中
- 支持 Promise、对象、数组、DOM 元素
- `.each()` 中必须用 `.wrap()` 包装元素
- 可以与 `.as()` 结合创建别名

## 12. 如何使用 .trigger() 触发自定义事件？

```javascript
cy.get('.element').trigger('mouseout');
```

**与其他命令对比**:

```javascript
// .click() vs .trigger('click')
```

- 可传递事件选项（坐标、修饰键等）
- 不执行可操作性检查（与 `.click()` 不同）
- 常用于不支持的操作或自定义组件
- 适合测试事件监听器

## 13. .scrollIntoView() 和 .scrollTo() 的区别和用法？

**答案**:

两个命令都用于滚动，但作用方式不同。

```javascript
// .scrollIntoView() - 滚动使元素可见
cy.get('.footer').scrollIntoView();
cy.get('#section-5').scrollIntoView().should('be.visible');

// .scrollTo() - 滚动容器到指定位置
cy.scrollTo(0, 500);                    // 页面滚动到 y=500
cy.scrollTo('bottom');                  // 滚动到底部
cy.scrollTo('top');                     // 滚动到顶部
cy.get('.container').scrollTo('center'); // 容器内滚动
```

**详细用法**:

```javascript
describe('滚动命令对比', () => {
  it('.scrollIntoView() 使用', () => {
    // 滚动到元素可见
    cy.get('#bottom-element').scrollIntoView();
    cy.get('#bottom-element').should('be.visible');

    // 带选项
    cy.get('.element').scrollIntoView({
      duration: 2000,      // 滚动时长
      easing: 'swing',     // 缓动函数
      offset: { top: -100, left: 0 }  // 偏移量
    });
  });

  it('.scrollTo() 使用', () => {
    // 预设位置
    cy.scrollTo('top');
    cy.scrollTo('bottom');
    cy.scrollTo('center');
    cy.scrollTo('topLeft');
    cy.scrollTo('bottomRight');

    // 坐标位置
    cy.scrollTo(0, 1000);
    cy.scrollTo(500, 500);

    // 百分比
    cy.scrollTo('0%', '50%');

    // 带选项
    cy.scrollTo('bottom', {
      duration: 2000,
      easing: 'linear'
    });
  });

  it('容器滚动', () => {
    // 滚动特定容器
    cy.get('.scrollable-div').scrollTo('bottom');

    // 横向滚动
    cy.get('.horizontal-scroll').scrollTo('right');
    cy.get('.horizontal-scroll').scrollTo(500, 0);
  });
});
```

**实际应用场景**:

```javascript

cy.get('#section-3').scrollIntoView().should('be.visible');

// 场景 2: 无限滚动加载

cy.wait('@loadMore');
cy.scrollTo('bottom');
cy.wait('@loadMore');

// 场景 3: 查找并滚动到元素

// 场景 4: 表格滚动
cy.get('table').scrollTo('bottomRight');
cy.get('table tbody tr').last().should('be.visible');

// 场景 5: 模态框滚动
cy.get('.modal-body').scrollTo('bottom');
cy.get('.modal-body .accept-btn').should('be.visible');

// 场景 6: 横向滚动容器
cy.get('.carousel').scrollTo('right');
cy.get('.carousel-item:last').should('be.visible');

// 场景 7: 逐步滚动
for (let i = 0; i < 5; i++) {
  cy.scrollTo(0, i * 500);
  cy.wait(500);
}

// 场景 8: 滚动到元素后操作
cy.get('.lazy-loaded-image')
  .scrollIntoView()
  .should('have.attr', 'src')
  .and('include', 'image.jpg');
```

**结合等待使用**:

```javascript

cy.wait(2000);
cy.get('.footer').should('be.visible');

// 等待懒加载
cy.get('.lazy-element')
  .scrollIntoView()
  .should('have.attr', 'data-loaded', 'true');

// 滚动并等待网络请求

cy.scrollTo('bottom');
cy.wait('@loadMore');
cy.get('.new-items').should('exist');
```

**注意事项**:

```javascript
// ❌ Cypress 自动滚动元素，通常不需要手动滚动
cy.get('.button').scrollIntoView().click();  // 不必要

// ✅ Cypress 自动滚动
cy.get('.button').click();  // 已自动滚动

// ✅ 仅在需要验证滚动行为时使用
cy.get('.element').scrollIntoView();
cy.window().its('scrollY').should('be.gt', 100);

// ✅ 懒加载等特殊场景
cy.get('.lazy-content').scrollIntoView();
cy.get('.lazy-content img').should('be.visible');
```

**关键点**:
- `.scrollIntoView()`: 滚动到特定元素
- `.scrollTo()`: 滚动到指定位置或方向
- Cypress 自动滚动元素，通常无需手动操作
- 适用于无限滚动、懒加载等场景
- 可设置滚动时长和缓动效果

## 14. .select() 命令如何使用？

**答案**:

`.select()` 用于选择 `<select>` 下拉选项。

```javascript
// 1. 通过文本选择
cy.get('select').select('Option 1');

// 2. 通过值选择
cy.get('select').select('option-1-value');

// 3. 通过索引选择
cy.get('select').select(2);  // 第 3 个选项（从 0 开始）

// 4. 多选
cy.get('select[multiple]').select(['option1', 'option2', 'option3']);

// 5. 带选项
cy.get('select').select('Option 1', { force: true });
```

**详细示例**:

```javascript
describe('select 命令使用', () => {
  beforeEach(() => {
    cy.visit('/form');
  });

  it('单选下拉', () => {
    // HTML: <select name="country">
    //         <option value="us">United States</option>
    //         <option value="uk">United Kingdom</option>
    //         <option value="ca">Canada</option>
    //       </select>

    // 方式 1: 通过显示文本
    cy.get('select[name="country"]')
      .select('Canada')
      .should('have.value', 'ca');

    // 方式 2: 通过 value 属性
    cy.get('select[name="country"]')
      .select('uk')
      .should('have.value', 'uk');

    // 方式 3: 通过索引
    cy.get('select[name="country"]')
      .select(0)
      .should('have.value', 'us');
  });

  it('多选下拉', () => {
    // HTML: <select name="skills" multiple>
    //         <option value="js">JavaScript</option>
    //         <option value="py">Python</option>
    //         <option value="java">Java</option>
    //       </select>

    cy.get('select[name="skills"]')
      .select(['js', 'py'])
      .invoke('val')
      .should('deep.equal', ['js', 'py']);
  });

  it('验证选项', () => {
    cy.get('select')
      .select('Option 2')
      .should('have.value', 'opt-2')
      .find('option:selected')
      .should('have.text', 'Option 2');
  });

  it('验证所有选项', () => {
    cy.get('select option').should('have.length', 5);

    cy.get('select option').then(($options) => {
      const values = Cypress._.map($options, 'value');
      expect(values).to.deep.eq(['opt1', 'opt2', 'opt3', 'opt4', 'opt5']);
    });
  });
});
```

**实际应用场景**:

**处理特殊情况**:


## 15. .check() 和 .uncheck() 如何使用？

**答案**:

`.check()` 和 `.uncheck()` 用于操作复选框和单选按钮。

```javascript
// 1. 选中复选框
cy.get('input[type="checkbox"]').check();

// 2. 取消选中
cy.get('input[type="checkbox"]').uncheck();

// 3. 选中单选按钮
cy.get('input[type="radio"]').check();

// 4. 通过值选中
cy.get('input[type="checkbox"]').check(['option1', 'option2']);
cy.get('input[type="radio"]').check('male');

// 5. 带选项
cy.get('input[type="checkbox"]').check({ force: true });
```

**详细示例**:

```javascript
describe('check/uncheck 命令', () => {
  it('复选框操作', () => {
    // 选中单个
    cy.get('#agree').check().should('be.checked');

    // 取消选中
    cy.get('#agree').uncheck().should('not.be.checked');

    // 选中多个
    cy.get('input[type="checkbox"]')
      .check()
      .should('be.checked');

    // 选中特定复选框
    cy.get('input[type="checkbox"]')
      .check(['option1', 'option3']);

    cy.get('#option1').should('be.checked');
    cy.get('#option2').should('not.be.checked');
    cy.get('#option3').should('be.checked');
  });

  it('单选按钮操作', () => {
    // 选中单选按钮
    cy.get('input[name="gender"][value="female"]')
      .check()
      .should('be.checked');

    // 验证其他按钮未选中
    cy.get('input[name="gender"][value="male"]')
      .should('not.be.checked');

    // 通过值选中
    cy.get('input[name="size"]').check('large');
    cy.get('input[name="size"][value="large"]')
      .should('be.checked');
  });

  it('条件性操作', () => {
    // 仅在未选中时选中
    cy.get('#option').then(($checkbox) => {
      if (!$checkbox.is(':checked')) {
        cy.get('#option').check();
      }
    });

    // 或直接使用，Cypress 会处理
    cy.get('#option').check();  // 已选中则不变
  });
});
```

**实际应用场景**:

```javascript

cy.get('button[type="submit"]').click();

```

**验证状态**:

```javascript
  .and('have.value', 'yes');

// 验证未选中

  .should('not.be.checked');

// 验证禁用状态

  .then(($checked) => {
    const values = Cypress._.map($checked, 'value');
    expect(values).to.deep.eq(['opt1', 'opt3']);
  });
```

**注意事项**:

- `.check()` 用于复选框和单选按钮

- 自动跳过已选中的元素
- 支持通过值选中特定元素
- 可批量操作多个元素
- 自动触发 `change` 事件
