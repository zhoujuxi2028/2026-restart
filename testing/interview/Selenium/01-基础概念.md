## 第 1 题：什么是 Selenium？它有哪些主要组件？

**答案**：

Selenium 是一个用于 Web 应用自动化测试的开源工具集，支持多种浏览器和编程语言。

**主要组件**：

1. **Selenium WebDriver**：核心组件，直接与浏览器通信
2. **Selenium IDE**：浏览器插件，用于录制和回放测试
3. **Selenium Grid**：分布式测试执行平台，支持并行测试

**关键点**：
- WebDriver 是最常用的组件
- 支持 Python、Java、C#、JavaScript 等多种语言
- 遵循 W3C WebDriver 标准协议

## 第 2 题：Selenium WebDriver 的工作原理是什么？

**答案**：

WebDriver 通过浏览器驱动程序与浏览器进行通信：

```
测试脚本 → WebDriver API → 浏览器驱动 → 浏览器
         ← JSON Wire Protocol ←          ←
```

**工作流程**：
1. 测试脚本调用 WebDriver API
2. WebDriver 将命令转换为 HTTP 请求
3. 浏览器驱动（如 ChromeDriver）接收请求
4. 驱动控制浏览器执行操作
5. 结果通过 HTTP 响应返回

**关键点**：
- 使用 JSON Wire Protocol 或 W3C WebDriver Protocol
- 每个浏览器需要对应的驱动程序
- 驱动程序是浏览器和测试代码之间的桥梁

## 第 3 题：如何初始化不同浏览器的 WebDriver？

**答案**：

```python
from selenium import webdriver
from selenium.webdriver.chrome.service import Service
from selenium.webdriver.chrome.options import Options

# 1. Chrome 浏览器
chrome_options = Options()
chrome_options.add_argument('--headless')  # 无头模式
driver = webdriver.Chrome(options=chrome_options)

# 2. Firefox 浏览器
firefox_options = webdriver.FirefoxOptions()
driver = webdriver.Firefox(options=firefox_options)

# 3. Edge 浏览器
driver = webdriver.Edge()

# 4. Safari 浏览器
driver = webdriver.Safari()

# 5. 使用 WebDriver Manager（自动管理驱动）
from webdriver_manager.chrome import ChromeDriverManager

driver = webdriver.Chrome(
    service=Service(ChromeDriverManager().install())
)
```

**关键点**：
- 推荐使用 `webdriver-manager` 自动管理驱动版本
- Options 用于配置浏览器启动参数
- 记得在测试结束后调用 `driver.quit()`

## 第 4 题：Selenium 3 和 Selenium 4 有什么主要区别？

**答案**：

| 特性 | Selenium 3 | Selenium 4 |
|------|-----------|-----------|
| **协议** | JSON Wire Protocol | W3C WebDriver Protocol |
| **驱动管理** | 需要手动下载 | 支持自动下�（Beta） |
| **相对定位** | 不支持 | 支持（above, below, near 等） |
| **CDP 支持** | 不支持 | 支持 Chrome DevTools Protocol |
| **窗口管理** | 基础功能 | 增强功能（新窗口、新标签） |
| **截图** | 元素截图需要第三方库 | 原生支持元素截图 |

**Selenium 4 代码示例**：

```python
from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.support.relative_locator import locate_with

driver = webdriver.Chrome()

# 1. 相对定位
password_field = driver.find_element(
    locate_with(By.TAG_NAME, "input").below({By.ID: "username"})
)

# 2. 元素截图
element = driver.find_element(By.ID, "logo")
element.screenshot("logo.png")

# 3. 打开新窗口
driver.switch_to.new_window('tab')
```

**关键点**：
- Selenium 4 完全兼容 W3C 标准
- 相对定位提高了定位的灵活性
- CDP 支持允许更底层的浏览器控制

## 第 6 题：什么是 WebElement？如何操作它？

**答案**：

WebElement 代表页面上的一个 DOM 元素，提供了丰富的操作方法。

```python
from selenium import webdriver
from selenium.webdriver.common.by import By

driver = webdriver.Chrome()
driver.get("https://www.example.com")

# 查找元素
element = driver.find_element(By.ID, "username")

# 1. 输入文本
element.send_keys("test_user")

# 2. 清空内容
element.clear()

# 3. 点击元素
element.click()

# 4. 获取属性
value = element.get_attribute("value")
class_name = element.get_attribute("class")

# 5. 获取文本
text = element.text

# 6. 检查元素状态
is_displayed = element.is_displayed()  # 是否可见
is_enabled = element.is_enabled()      # 是否可用
is_selected = element.is_selected()    # 是否被选中（checkbox/radio）

# 7. 获取元素位置和大小
location = element.location  # {'x': 100, 'y': 200}
size = element.size          # {'width': 300, 'height': 50}

# 8. 获取 CSS 属性
color = element.value_of_css_property("color")
```

**关键点**：
- 元素操作前应确保元素可见和可交互
- `text` 获取的是可见文本，不包括隐藏元素
- 属性值可能为 `None`

## 第 7 题：如何配置浏览器选项（ChromeOptions）？

**答案**：

```python
from selenium import webdriver
from selenium.webdriver.chrome.options import Options

options = Options()

# 1. 无头模式（不显示浏览器窗口）
options.add_argument('--headless')
options.add_argument('--disable-gpu')

# 2. 窗口大小
options.add_argument('--window-size=1920,1080')
options.add_argument('--start-maximized')  # 最大化

# 3. 禁用通知和弹窗
options.add_argument('--disable-notifications')
options.add_argument('--disable-popup-blocking')

# 4. 禁用图片加载（提升速度）
prefs = {
    'profile.default_content_setting_values': {
        'images': 2,
        'javascript': 1
    }
}
options.add_experimental_option('prefs', prefs)

# 5. 设置 User-Agent
options.add_argument('user-agent=Mozilla/5.0 ...')

# 6. 忽略证书错误
options.add_argument('--ignore-certificate-errors')

# 7. 禁用自动化标志（反爬虫）
options.add_experimental_option('excludeSwitches', ['enable-automation'])
options.add_experimental_option('useAutomationExtension', False)

# 8. 设置下载路径
options.add_experimental_option('prefs', {
    'download.default_directory': '/path/to/download',
    'download.prompt_for_download': False
})

# 9. 开启性能日志
options.set_capability('goog:loggingPrefs', {'performance': 'ALL'})

driver = webdriver.Chrome(options=options)
```

**关键点**：
- 无头模式常用于 CI/CD 环境
- 反爬虫检测可通过配置绕过部分检测
- 性能日志用于分析网络请求

## 第 8 题：如何执行 JavaScript 代码？

**答案**：

```python
from selenium import webdriver
from selenium.webdriver.common.by import By

driver = webdriver.Chrome()
driver.get("https://www.example.com")

# 1. 执行简单脚本
driver.execute_script("alert('Hello Selenium');")

# 2. 返回值
title = driver.execute_script("return document.title;")
print(title)

# 3. 操作元素
element = driver.find_element(By.ID, "username")
driver.execute_script("arguments[0].style.border='3px solid red';", element)

# 4. 滚动到元素
element = driver.find_element(By.ID, "footer")
driver.execute_script("arguments[0].scrollIntoView();", element)

# 5. 滚动到页面底部
driver.execute_script("window.scrollTo(0, document.body.scrollHeight);")

# 6. 修改元素属性
driver.execute_script(
    "arguments[0].setAttribute('value', 'new value');",
    element
)

# 7. 移除元素属性（如 readonly）
driver.execute_script("arguments[0].removeAttribute('readonly');", element)

# 8. 点击被遮挡的元素
driver.execute_script("arguments[0].click();", element)

# 9. 获取本地存储
local_storage = driver.execute_script("return window.localStorage;")

# 10. 异步执行脚本
result = driver.execute_async_script("""
    var callback = arguments[arguments.length - 1];
    setTimeout(function() {
        callback('Async result');
    }, 1000);
""")
```

**关键点**：
- `arguments[0]` 代表传入的第一个参数
- JavaScript 可以绕过某些 Selenium 限制
- 异步脚本需要调用 callback

## 第 9 题：如何处理浏览器 Cookie？

**答案**：

```python
from selenium import webdriver
import time

driver = webdriver.Chrome()
driver.get("https://www.example.com")

# 1. 获取所有 Cookie
all_cookies = driver.get_cookies()
print(all_cookies)

# 2. 获取指定 Cookie
session_cookie = driver.get_cookie("session_id")
if session_cookie:
    print(f"Session ID: {session_cookie['value']}")

# 3. 添加 Cookie
driver.add_cookie({
    'name': 'test_cookie',
    'value': 'test_value',
    'path': '/',
    'domain': '.example.com',
    'secure': True,
    'httpOnly': False,
    'expiry': int(time.time()) + 3600  # 1小时后过期
})

# 4. 删除指定 Cookie
driver.delete_cookie("test_cookie")

# 5. 删除所有 Cookie
driver.delete_all_cookies()

# 6. Cookie 持久化（保存登录状态）
import pickle

# 登录后保存 Cookie
driver.get("https://www.example.com/login")
# ... 执行登录操作 ...
pickle.dump(driver.get_cookies(), open("cookies.pkl", "wb"))

# 下次直接加载 Cookie
driver.get("https://www.example.com")
cookies = pickle.load(open("cookies.pkl", "rb"))
for cookie in cookies:
    driver.add_cookie(cookie)
driver.refresh()  # 刷新页面使 Cookie 生效
```

**关键点**：
- Cookie 必须在访问对应域名后才能添加
- `domain` 需要匹配当前页面域名
- Cookie 持久化可以避免重复登录

## 第 10 题：如何处理浏览器警告框（Alert）？

**答案**：

```python
from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC
from selenium.common.exceptions import NoAlertPresentException

driver = webdriver.Chrome()
driver.get("https://www.example.com")

# 触发 Alert
driver.find_element(By.ID, "alert-button").click()

# 等待 Alert 出现
WebDriverWait(driver, 10).until(EC.alert_is_present())

# 1. 切换到 Alert
alert = driver.switch_to.alert

# 2. 获取 Alert 文本
alert_text = alert.text
print(f"Alert 内容: {alert_text}")

# 3. 接受 Alert（点击确定）
alert.accept()

# 4. 拒绝 Alert（点击取消）
# alert.dismiss()

# 5. 输入文本（针对 prompt 类型）
driver.find_element(By.ID, "prompt-button").click()
WebDriverWait(driver, 10).until(EC.alert_is_present())
alert = driver.switch_to.alert
alert.send_keys("输入的文本")
alert.accept()

# 6. 检查是否存在 Alert
def is_alert_present():
    try:
        driver.switch_to.alert
        return True
    except NoAlertPresentException:
        return False

if is_alert_present():
    driver.switch_to.alert.accept()
```

**Alert 类型**：

| 类型 | 方法 | 说明 |
|------|------|------|
| **alert** | `accept()` | 提示框，只有确定按钮 |
| **confirm** | `accept()` / `dismiss()` | 确认框，有确定和取消 |
| **prompt** | `send_keys()` + `accept()` | 输入框，可输入文本 |

**关键点**：
- Alert 出现时会阻塞页面操作
- 必须处理 Alert 才能继续操作页面
- 使用等待机制确保 Alert 已加载

## 第 11 题：如何处理多窗口和标签页切换？

**答案**：

```python
from selenium import webdriver
from selenium.webdriver.common.by import By

driver = webdriver.Chrome()
driver.get("https://www.example.com")

# 1. 获取当前窗口句柄
main_window = driver.current_window_handle
print(f"主窗口句柄: {main_window}")

# 2. 点击链接打开新窗口
driver.find_element(By.LINK_TEXT, "Open New Window").click()

# 3. 获取所有窗口句柄
all_windows = driver.window_handles
print(f"所有窗口: {all_windows}")

# 4. 切换到新窗口
for window in all_windows:
    if window != main_window:
        driver.switch_to.window(window)
        print(f"切换到窗口: {driver.title}")
        break

# 5. 在新窗口中操作
driver.find_element(By.ID, "search").send_keys("test")

# 6. 关闭当前窗口
driver.close()

# 7. 切换回主窗口
driver.switch_to.window(main_window)

# 8. Selenium 4：打开新标签页
driver.switch_to.new_window('tab')
driver.get("https://www.google.com")

# 9. Selenium 4：打开新窗口
driver.switch_to.new_window('window')
driver.get("https://www.github.com")

# 10. 遍历所有窗口
for handle in driver.window_handles:
    driver.switch_to.window(handle)
    print(f"窗口标题: {driver.title}")
```

**关键点**：
- 窗口句柄是唯一标识符
- 新窗口打开后需要手动切换
- 关闭窗口后要切换回有效窗口
- Selenium 4 提供了更简便的新窗口 API

## 第 12 题：如何处理 iframe 和 frame？

**答案**：

```python
from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC

driver = webdriver.Chrome()
driver.get("https://www.example.com")

# 1. 通过索引切换
driver.switch_to.frame(0)  # 切换到第一个 frame

# 2. 通过 name 或 id 切换
driver.switch_to.frame("frame_name")

# 3. 通过 WebElement 切换
frame_element = driver.find_element(By.ID, "myframe")
driver.switch_to.frame(frame_element)

# 4. 等待 frame 可用并切换
WebDriverWait(driver, 10).until(
    EC.frame_to_be_available_and_switch_to_it((By.ID, "myframe"))
)

# 5. 在 frame 中操作元素
driver.find_element(By.ID, "username").send_keys("test")

# 6. 切换回主文档
driver.switch_to.default_content()

# 7. 切换到父 frame
driver.switch_to.parent_frame()

# 8. 嵌套 frame 处理
driver.switch_to.frame("outer_frame")
driver.switch_to.frame("inner_frame")
# 操作嵌套 frame 中的元素
driver.find_element(By.ID, "nested_element").click()
# 回到顶层
driver.switch_to.default_content()
```

**Frame 层级结构**：

```
主文档 (default_content)
├── Frame A
│   └── Frame A1 (嵌套)
└── Frame B
```

**关键点**：
- 必须切换到 frame 才能操作其中的元素
- `default_content()` 返回顶层文档
- `parent_frame()` 返回上一层 frame
- 嵌套 frame 需要逐层切换

## 第 13 题：如何获取浏览器日志和性能数据？

**答案**：

```python
from selenium import webdriver
from selenium.webdriver.chrome.options import Options
import json

# 配置日志选项
options = Options()
options.set_capability('goog:loggingPrefs', {
    'browser': 'ALL',
    'performance': 'ALL'
})

driver = webdriver.Chrome(options=options)
driver.get("https://www.example.com")

# 1. 获取浏览器控制台日志
browser_logs = driver.get_log('browser')
for log in browser_logs:
    print(f"[{log['level']}] {log['message']}")

# 2. 获取性能日志（网络请求）
performance_logs = driver.get_log('performance')
for log in performance_logs:
    message = json.loads(log['message'])
    method = message['message']['method']

    # 过滤网络响应
    if method == 'Network.responseReceived':
        response = message['message']['params']['response']
        print(f"URL: {response['url']}")
        print(f"状态码: {response['status']}")

# 3. 使用 CDP 获取网络请求
driver.execute_cdp_cmd('Network.enable', {})
logs = driver.get_log('performance')

# 4. 获取页面加载时间
navigation_start = driver.execute_script(
    "return window.performance.timing.navigationStart"
)
load_complete = driver.execute_script(
    "return window.performance.timing.loadEventEnd"
)
load_time = (load_complete - navigation_start) / 1000
print(f"页面加载时间: {load_time}秒")

# 5. 获取资源加载信息
resources = driver.execute_script(
    "return window.performance.getEntriesByType('resource')"
)
for resource in resources:
    print(f"{resource['name']}: {resource['duration']}ms")
```

**日志类型**：

| 类型 | 说明 |
|------|------|
| `browser` | 浏览器控制台日志（console.log、错误等） |
| `performance` | 性能日志（网络请求、资源加载） |
| `driver` | WebDriver 操作日志 |

**关键点**：
- 日志需要在浏览器启动前配置
- 性能日志包含详细的网络请求信息
- 可用于性能测试和接口监控

## 第 15 题：如何设置页面加载超时和脚本超时？

**答案**：

```python
from selenium import webdriver
from selenium.common.exceptions import TimeoutException

driver = webdriver.Chrome()

# 1. 页面加载超时（秒）
driver.set_page_load_timeout(30)

# 2. 脚本执行超时（秒）
driver.set_script_timeout(10)

# 3. 隐式等待（全局元素查找超时）
driver.implicitly_wait(10)

# 测试页面加载超时
try:
    driver.get("https://very-slow-website.com")
except TimeoutException:
    print("页面加载超时")
    driver.execute_script("window.stop();")  # 停止加载

# 测试脚本超时
try:
    result = driver.execute_async_script("""
        var callback = arguments[arguments.length - 1];
        setTimeout(function() {
            callback('Done');
        }, 15000);  # 15秒后返回，但超时设置为10秒
    """)
except TimeoutException:
    print("脚本执行超时")

# 最佳实践：综合配置
class DriverConfig:
    def __init__(self):
        self.driver = webdriver.Chrome()
        self._configure_timeouts()

    def _configure_timeouts(self):
        # 页面加载最多等待30秒
        self.driver.set_page_load_timeout(30)
        # 脚本执行最多10秒
        self.driver.set_script_timeout(10)
        # 元素查找默认等待10秒（不推荐，建议用显式等待）
        # self.driver.implicitly_wait(10)
```

**超时类型对比**：

| 超时类型 | 方法 | 作用范围 | 推荐值 |
|---------|------|---------|--------|
| **页面加载超时** | `set_page_load_timeout()` | `driver.get()` | 30秒 |
| **脚本执行超时** | `set_script_timeout()` | `execute_async_script()` | 10秒 |
| **隐式等待** | `implicitly_wait()` | 所有 `find_element()` | 不推荐使用 |

**关键点**：
