## 第 1 题：Selenium 有哪些元素定位方法？

**答案**：

Selenium 提供了 8 种基本定位方法（通过 `By` 类）：

```python
from selenium import webdriver
from selenium.webdriver.common.by import By

driver = webdriver.Chrome()
driver.get("https://www.example.com")

# 1. ID 定位（最快，最优先）
element = driver.find_element(By.ID, "username")

# 2. NAME 定位
element = driver.find_element(By.NAME, "email")

# 3. CLASS_NAME 定位
element = driver.find_element(By.CLASS_NAME, "btn-primary")

# 4. TAG_NAME 定位
element = driver.find_element(By.TAG_NAME, "input")

# 5. LINK_TEXT 定位（完全匹配链接文本）
element = driver.find_element(By.LINK_TEXT, "登录")

# 6. PARTIAL_LINK_TEXT 定位（部分匹配）
element = driver.find_element(By.PARTIAL_LINK_TEXT, "登")

# 7. CSS_SELECTOR 定位（强大灵活）
element = driver.find_element(By.CSS_SELECTOR, "#username")

# 8. XPATH 定位（最强大但最慢）
element = driver.find_element(By.XPATH, "//input[@id='username']")
```

**定位方法优先级**：

```
ID > NAME > CSS_SELECTOR > XPATH > CLASS_NAME > TAG_NAME > LINK_TEXT
```

**关键点**：
- ID 是最快且最稳定的定位方式
- CSS Selector 比 XPath 性能更好
- 避免使用 TAG_NAME 单独定位（太泛）

## 第 2 题：CSS Selector 和 XPath 有什么区别？如何选择？

**答案**：

**对比表**：

| 特性 | CSS Selector | XPath |
|------|-------------|-------|
| **性能** | 快 | 较慢 |
| **语法** | 简洁 | 复杂 |
| **浏览器支持** | 原生支持 | 需要解析 |
| **方向** | 只能向下查找 | 可以向上查找父元素 |
| **文本定位** | 不直接支持 | 支持（contains、text()） |
| **索引** | 从 1 开始 | 从 1 开始 |

**CSS Selector 示例**：

```python
# 1. ID 选择器
driver.find_element(By.CSS_SELECTOR, "#username")

# 2. Class 选择器
driver.find_element(By.CSS_SELECTOR, ".btn-primary")

# 3. 属性选择器
driver.find_element(By.CSS_SELECTOR, "input[name='email']")
driver.find_element(By.CSS_SELECTOR, "input[type='password']")

# 4. 后代选择器
driver.find_element(By.CSS_SELECTOR, "div.form input#username")

# 5. 子选择器
driver.find_element(By.CSS_SELECTOR, "ul > li:first-child")

# 6. 伪类选择器
driver.find_element(By.CSS_SELECTOR, "li:nth-child(2)")
driver.find_element(By.CSS_SELECTOR, "input:disabled")

# 7. 多属性组合
driver.find_element(By.CSS_SELECTOR, "input[type='text'][name='username']")

# 8. 部分匹配
driver.find_element(By.CSS_SELECTOR, "a[href*='login']")  # 包含
driver.find_element(By.CSS_SELECTOR, "a[href^='https']")  # 开头
driver.find_element(By.CSS_SELECTOR, "a[href$='.pdf']")   # 结尾
```

**XPath 示例**：

```python
# 1. 绝对路径（不推荐）
driver.find_element(By.XPATH, "/html/body/div/form/input")

# 2. 相对路径（推荐）
driver.find_element(By.XPATH, "//input[@id='username']")

# 3. 文本定位
driver.find_element(By.XPATH, "//button[text()='登录']")
driver.find_element(By.XPATH, "//a[contains(text(),'注册')]")

# 4. 多属性组合
driver.find_element(By.XPATH, "//input[@type='text' and @name='username']")

# 5. 向上查找父元素（CSS 不支持）
driver.find_element(By.XPATH, "//input[@id='username']/..")

# 6. 查找兄弟元素
driver.find_element(By.XPATH, "//label[@for='username']/following-sibling::input")

# 7. 索引定位
driver.find_element(By.XPATH, "(//input[@type='text'])[2]")

# 8. 轴定位
driver.find_element(By.XPATH, "//div[@class='form']//ancestor::form")
```

**选择建议**：
- 优先使用 CSS Selector（性能更好）
- 需要向上查找或文本匹配时使用 XPath
- 复杂层级建议用 CSS

## 第 3 题：如何定位动态 ID 或动态属性的元素？

**答案**：

```python
from selenium import webdriver
from selenium.webdriver.common.by import By

driver = webdriver.Chrome()
driver.get("https://www.example.com")

# 场景：ID 是动态生成的，如 "user_12345"、"user_67890"

# 方法 1：CSS 部分匹配
# ID 以 "user_" 开头
driver.find_element(By.CSS_SELECTOR, "[id^='user_']")
# ID 以 "_submit" 结尾
driver.find_element(By.CSS_SELECTOR, "[id$='_submit']")
# ID 包含 "user"
driver.find_element(By.CSS_SELECTOR, "[id*='user']")

# 方法 2：XPath 部分匹配
driver.find_element(By.XPATH, "//input[starts-with(@id, 'user_')]")
driver.find_element(By.XPATH, "//button[contains(@id, 'submit')]")

# 方法 3：通过其他稳定属性定位
driver.find_element(By.CSS_SELECTOR, "input[name='username']")
driver.find_element(By.CSS_SELECTOR, "input[type='text'][placeholder='用户名']")

# 方法 4：通过父元素或兄弟元素定位
# 先找到稳定的父元素，再找子元素
parent = driver.find_element(By.CLASS_NAME, "login-form")
input_field = parent.find_element(By.TAG_NAME, "input")

# 方法 5：XPath 轴定位
# 通过 label 找到 input
driver.find_element(
    By.XPATH,
    "//label[text()='用户名']/following-sibling::input"
)

# 方法 6：组合多个稳定属性
driver.find_element(
    By.CSS_SELECTOR,
    "input[type='text'][placeholder='请输入用户名'][class='form-control']"
)
```

**最佳实践**：

```python
# 创建健壮的定位策略
class RobustLocator:
    @staticmethod
    def find_element_with_fallback(driver, locators):
        """按优先级尝试多个定位器"""
        for by, value in locators:
            try:
                element = driver.find_element(by, value)
                if element.is_displayed():
                    return element
            except:
                continue
        raise Exception("所有定位策略都失败")

# 使用示例
locators = [
    (By.ID, "username"),
    (By.NAME, "username"),
    (By.CSS_SELECTOR, "input[placeholder='用户名']"),
    (By.XPATH, "//input[@type='text'][1]")
]
element = RobustLocator.find_element_with_fallback(driver, locators)
```

**关键点**：
- 动态属性要找不变的部分
- 优先使用稳定的属性（name、class）
- 多个定位策略作为备选方案

## 第 4 题：如何定位表格中的元素？

**答案**：

```python
from selenium import webdriver
from selenium.webdriver.common.by import By

driver = webdriver.Chrome()
driver.get("https://www.example.com/table")

# HTML 表格结构：
# <table id="data-table">
#   <tr><th>姓名</th><th>年龄</th><th>操作</th></tr>
#   <tr><td>张三</td><td>25</td><td><button>编辑</button></td></tr>
#   <tr><td>李四</td><td>30</td><td><button>编辑</button></td></tr>
# </table>

# 1. 获取表格所有行
rows = driver.find_elements(By.XPATH, "//table[@id='data-table']//tr")
print(f"总行数: {len(rows)}")

# 2. 获取第 2 行第 3 列的数据（索引从 1 开始）
cell = driver.find_element(
    By.XPATH,
    "//table[@id='data-table']//tr[2]/td[3]"
)

# 3. 获取特定单元格（CSS）
cell = driver.find_element(
    By.CSS_SELECTOR,
    "#data-table tr:nth-child(2) td:nth-child(3)"
)

# 4. 根据文本内容定位行
row = driver.find_element(
    By.XPATH,
    "//table[@id='data-table']//tr[td[text()='张三']]"
)

# 5. 找到包含"张三"的行，点击该行的"编辑"按钮
driver.find_element(
    By.XPATH,
    "//tr[td[text()='张三']]//button[text()='编辑']"
).click()

# 6. 遍历表格所有数据
table = driver.find_element(By.ID, "data-table")
rows = table.find_elements(By.TAG_NAME, "tr")

for i, row in enumerate(rows[1:], 1):  # 跳过表头
    cells = row.find_elements(By.TAG_NAME, "td")
    name = cells[0].text
    age = cells[1].text
    print(f"第{i}行: {name}, {age}岁")

# 7. 获取表头
headers = driver.find_elements(
    By.XPATH,
    "//table[@id='data-table']//th"
)
header_texts = [h.text for h in headers]
print(f"表头: {header_texts}")

# 8. 根据列名获取数据（需要先找到列索引）
def get_column_index(driver, table_id, column_name):
    headers = driver.find_elements(
        By.XPATH,
        f"//table[@id='{table_id}']//th"
    )
    for i, header in enumerate(headers, 1):
        if header.text == column_name:
            return i
    return None

# 获取"年龄"列的索引
age_index = get_column_index(driver, "data-table", "年龄")
# 获取第 2 行的年龄
age = driver.find_element(
    By.XPATH,
    f"//table[@id='data-table']//tr[2]/td[{age_index}]"
).text

# 9. 动态表格（支持分页）
def get_table_data(driver, table_id):
    data = []
    rows = driver.find_elements(
        By.XPATH,
        f"//table[@id='{table_id}']//tbody/tr"
    )
    for row in rows:
        cells = row.find_elements(By.TAG_NAME, "td")
        row_data = [cell.text for cell in cells]
        data.append(row_data)
    return data

table_data = get_table_data(driver, "data-table")
```

**复杂表格定位技巧**：

```python
# 10. 跨行跨列的单元格（rowspan/colspan）
cell = driver.find_element(
    By.XPATH,
    "//td[@rowspan='2' and contains(text(),'合计')]"
)

# 11. 嵌套表格
outer_cell = driver.find_element(
    By.XPATH,
    "//table[@id='outer']//tr[1]/td[1]"
)
inner_table = outer_cell.find_element(By.TAG_NAME, "table")
inner_cell = inner_table.find_element(
    By.XPATH,
    ".//tr[1]/td[1]"
)
```

**关键点**：
- XPath 索引从 1 开始
- 优先根据文本内容定位行
- 遍历表格时注意跳过表头

## 第 5 题：如何定位下拉列表（Select）中的选项？

**答案**：

```python
from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.support.ui import Select

driver = webdriver.Chrome()
driver.get("https://www.example.com")

# HTML: <select id="country">
#         <option value="cn">中国</option>
#         <option value="us">美国</option>
#         <option value="uk">英国</option>
#       </select>

# 1. 创建 Select 对象
select_element = driver.find_element(By.ID, "country")
select = Select(select_element)

# 2. 通过索引选择（从 0 开始）
select.select_by_index(1)  # 选择第 2 个选项（美国）

# 3. 通过 value 属性选择
select.select_by_value("cn")  # 选择 value="cn" 的选项

# 4. 通过可见文本选择
select.select_by_visible_text("英国")

# 5. 获取所有选项
all_options = select.options
for option in all_options:
    print(f"Value: {option.get_attribute('value')}, Text: {option.text}")

# 6. 获取当前选中的选项
selected_option = select.first_selected_option
print(f"当前选中: {selected_option.text}")

# 7. 获取所有选中的选项（多选）
all_selected = select.all_selected_options

# 8. 取消选择（仅多选支持）
# select.deselect_by_index(0)
# select.deselect_by_value("cn")
# select.deselect_by_visible_text("中国")
# select.deselect_all()

# 9. 检查是否是多选下拉框
is_multiple = select.is_multiple
print(f"是否多选: {is_multiple}")

# 10. 非 <select> 标签的下拉列表（自定义下拉）
# 例如：Bootstrap、Ant Design 等
driver.find_element(By.ID, "custom-select").click()
driver.find_element(By.XPATH, "//li[@data-value='option1']").click()

# 11. 搜索型下拉框
search_select = driver.find_element(By.CLASS_NAME, "select2-search__field")
search_select.send_keys("北京")
driver.find_element(By.XPATH, "//li[contains(text(),'北京市')]").click()
```

**多选下拉框**：

```python
# HTML: <select id="skills" multiple>
#         <option value="python">Python</option>
#         <option value="java">Java</option>
#         <option value="js">JavaScript</option>
#       </select>

select = Select(driver.find_element(By.ID, "skills"))

# 选择多个选项
select.select_by_value("python")
select.select_by_value("java")

# 获取所有选中项
selected = select.all_selected_options
print([opt.text for opt in selected])  # ['Python', 'Java']

# 取消所有选择
select.deselect_all()
```

**关键点**：
- Select 类只适用于 `<select>` 标签
- 自定义下拉框需要模拟点击操作
- 多选下拉框需要 `multiple` 属性

## 第 6 题：如何定位列表中的第 n 个元素？

**答案**：

```python
from selenium import webdriver
from selenium.webdriver.common.by import By

driver = webdriver.Chrome()
driver.get("https://www.example.com")

# HTML: <ul id="list">
#         <li>Item 1</li>
#         <li>Item 2</li>
#         <li>Item 3</li>
#       </ul>

# 方法 1：XPath 索引定位（从 1 开始）
second_item = driver.find_element(
    By.XPATH,
    "//ul[@id='list']/li[2]"
)

# 方法 2：CSS 伪类选择器
second_item = driver.find_element(
    By.CSS_SELECTOR,
    "#list li:nth-child(2)"
)

# 方法 3：first-child 和 last-child
first_item = driver.find_element(
    By.CSS_SELECTOR,
    "#list li:first-child"
)
last_item = driver.find_element(
    By.CSS_SELECTOR,
    "#list li:last-child"
)

# 方法 4：find_elements 获取列表后用索引（从 0 开始）
items = driver.find_elements(By.CSS_SELECTOR, "#list li")
second_item = items[1]  # 第 2 个
last_item = items[-1]   # 最后一个

# 方法 5：XPath 获取最后一个元素
last_item = driver.find_element(
    By.XPATH,
    "(//ul[@id='list']/li)[last()]"
)

# 方法 6：nth-of-type
second_li = driver.find_element(
    By.CSS_SELECTOR,
    "#list li:nth-of-type(2)"
)

# 方法 7：倒数第 n 个
# 倒数第 2 个
second_last = driver.find_element(
    By.CSS_SELECTOR,
    "#list li:nth-last-child(2)"
)

# 方法 8：偶数/奇数元素
even_items = driver.find_elements(
    By.CSS_SELECTOR,
    "#list li:nth-child(even)"
)
odd_items = driver.find_elements(
    By.CSS_SELECTOR,
    "#list li:nth-child(odd)"
)

# 实际应用：点击列表中第 3 个链接
driver.find_element(
    By.CSS_SELECTOR,
    "ul#menu li:nth-child(3) a"
).click()
```

**CSS nth-child 语法**：

```python
# 选择第 3 个
":nth-child(3)"

# 选择奇数位（1, 3, 5...）
":nth-child(odd)"

# 选择偶数位（2, 4, 6...）
":nth-child(even)"

# 选择前 3 个
":nth-child(-n+3)"

# 从第 3 个开始
":nth-child(n+3)"

# 每 3 个选一个
":nth-child(3n)"
```

**关键点**：
- XPath 索引从 1 开始，Python 列表索引从 0 开始
- CSS `:nth-child()` 从 1 开始计数
- 优先用 CSS Selector，性能更好

## 第 7 题：如何定位影子 DOM（Shadow DOM）中的元素？

**答案**：

```python
from selenium import webdriver
from selenium.webdriver.common.by import By

driver = webdriver.Chrome()
driver.get("https://www.example.com")

# Shadow DOM 结构：
# <div id="host">
#   #shadow-root
#     <input id="shadow-input">
# </div>

# 方法 1：使用 JavaScript 访问 Shadow Root
host_element = driver.find_element(By.ID, "host")
shadow_root = driver.execute_script(
    "return arguments[0].shadowRoot",
    host_element
)

# 在 Shadow Root 中查找元素
shadow_input = shadow_root.find_element(By.ID, "shadow-input")
shadow_input.send_keys("test")

# 方法 2：Selenium 4 原生支持
# Selenium 4.0+ 提供了 shadow_root 属性
shadow_root = host_element.shadow_root
shadow_input = shadow_root.find_element(By.CSS_SELECTOR, "#shadow-input")

# 方法 3：嵌套 Shadow DOM
# <div id="host1">
#   #shadow-root
#     <div id="host2">
#       #shadow-root
#         <input id="nested-input">

host1 = driver.find_element(By.ID, "host1")
shadow_root1 = host1.shadow_root

host2 = shadow_root1.find_element(By.ID, "host2")
shadow_root2 = host2.shadow_root

nested_input = shadow_root2.find_element(By.ID, "nested-input")

# 方法 4：封装 Shadow DOM 操作
class ShadowDOMHelper:
    def __init__(self, driver):
        self.driver = driver

    def get_shadow_root(self, host_element):
        """获取元素的 Shadow Root"""
        return self.driver.execute_script(
            "return arguments[0].shadowRoot",
            host_element
        )

    def find_in_shadow(self, host_selector, shadow_selector):
        """在 Shadow DOM 中查找元素"""
        host = self.driver.find_element(By.CSS_SELECTOR, host_selector)
        shadow_root = self.get_shadow_root(host)
        return shadow_root.find_element(By.CSS_SELECTOR, shadow_selector)

# 使用示例
helper = ShadowDOMHelper(driver)
element = helper.find_in_shadow("#host", "#shadow-input")
element.send_keys("test")
```

**常见 Shadow DOM 场景**：

```python
# 1. Chrome 下载页面（chrome://downloads/）
host = driver.find_element(By.TAG_NAME, "downloads-manager")
shadow_root = host.shadow_root
items = shadow_root.find_elements(By.CSS_SELECTOR, "downloads-item")

# 2. Chrome 扩展页面（chrome://extensions/）
host = driver.find_element(By.TAG_NAME, "extensions-manager")
shadow_root1 = host.shadow_root
item_list = shadow_root1.find_element(By.ID, "items-list")
shadow_root2 = item_list.shadow_root
extensions = shadow_root2.find_elements(By.TAG_NAME, "extensions-item")
```

**关键点**：
- Shadow DOM 隔离了内部结构，常规定位无法访问
- Selenium 4 提供了原生 `shadow_root` 属性
- 嵌套 Shadow DOM 需要逐层获取

## 第 8 题：如何定位文件上传输入框并上传文件？

**答案**：

```python
from selenium import webdriver
from selenium.webdriver.common.by import By
import os

driver = webdriver.Chrome()
driver.get("https://www.example.com/upload")

# HTML: <input type="file" id="file-upload">

# 方法 1：直接发送文件路径（最简单）
file_input = driver.find_element(By.ID, "file-upload")
file_path = "/home/user/test.txt"
file_input.send_keys(file_path)

# 方法 2：上传多个文件（multiple 属性）
# HTML: <input type="file" id="multi-upload" multiple>
file_input = driver.find_element(By.ID, "multi-upload")
file1 = "/home/user/file1.txt"
file2 = "/home/user/file2.txt"
file_input.send_keys(f"{file1}\n{file2}")

# 方法 3：使用绝对路径
file_path = os.path.abspath("test.txt")
file_input.send_keys(file_path)

# 方法 4：隐藏的文件输入框
# 先用 JS 显示元素
file_input = driver.find_element(By.ID, "hidden-file-input")
driver.execute_script("arguments[0].style.display = 'block';", file_input)
file_input.send_keys(file_path)

# 方法 5：点击按钮触发文件选择（需要处理系统对话框）
# 这种情况推荐用 AutoIT（Windows）或 pyautogui
upload_button = driver.find_element(By.ID, "upload-button")
upload_button.click()
# 这里会弹出系统文件选择对话框，Selenium 无法直接处理
# 需要借助第三方工具

# 方法 6：验证文件已选择
file_input = driver.find_element(By.ID, "file-upload")
file_input.send_keys(file_path)
selected_file = file_input.get_attribute("value")
print(f"已选择文件: {selected_file}")

# 方法 7：完整上传流程
def upload_file(driver, file_input_id, file_path, submit_button_id):
    # 选择文件
    file_input = driver.find_element(By.ID, file_input_id)
    file_input.send_keys(file_path)

    # 点击上传按钮
    submit_button = driver.find_element(By.ID, submit_button_id)
    submit_button.click()

    # 等待上传完成
    from selenium.webdriver.support.ui import WebDriverWait
    from selenium.webdriver.support import expected_conditions as EC

    success_msg = WebDriverWait(driver, 10).until(
        EC.presence_of_element_located((By.CLASS_NAME, "upload-success"))
    )
    return success_msg.text

# 使用
result = upload_file(driver, "file-upload", "/path/to/file.txt", "submit-btn")
print(result)
```

**注意事项**：

```python
# 1. 远程 WebDriver（Selenium Grid）需要使用 LocalFileDetector
from selenium.webdriver.remote.file_detector import LocalFileDetector

driver.file_detector = LocalFileDetector()
file_input.send_keys("/local/path/to/file.txt")
```

## 第 9 题：Selenium 4 的相对定位（Relative Locators）如何使用？

**答案**：

Selenium 4 提供了基于空间位置的相对定位方法，非常适合定位表单元素。

```python
from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.support.relative_locator import locate_with

driver = webdriver.Chrome()
driver.get("https://www.example.com/form")

# 假设页面结构：
# <label>用户名：</label>
# <input id="username">
# <input id="password">
# <button>登录</button>

# 1. above()：在某元素上方
password_field = driver.find_element(By.ID, "password")
username_field = driver.find_element(
    locate_with(By.TAG_NAME, "input").above(password_field)
)

# 2. below()：在某元素下方
username_field = driver.find_element(By.ID, "username")
password_field = driver.find_element(
    locate_with(By.TAG_NAME, "input").below(username_field)
)

# 3. to_left_of()：在某元素左侧
submit_button = driver.find_element(By.ID, "submit")
cancel_button = driver.find_element(
    locate_with(By.TAG_NAME, "button").to_left_of(submit_button)
)

# 4. to_right_of()：在某元素右侧
cancel_button = driver.find_element(By.ID, "cancel")
submit_button = driver.find_element(
    locate_with(By.TAG_NAME, "button").to_right_of(cancel_button)
)

# 5. near()：在某元素附近（默认 50 像素内）
label = driver.find_element(By.XPATH, "//label[text()='用户名：']")
username_field = driver.find_element(
    locate_with(By.TAG_NAME, "input").near(label)
)

# 6. 组合使用
# 查找在 label 右侧且在 password 上方的 input
label = driver.find_element(By.XPATH, "//label[text()='用户名：']")
password_field = driver.find_element(By.ID, "password")
username_field = driver.find_element(
    locate_with(By.TAG_NAME, "input")
    .to_right_of(label)
    .above(password_field)
)

# 7. 使用字典形式传递元素
username_field = driver.find_element(
    locate_with(By.TAG_NAME, "input").below({By.ID: "email"})
)

# 8. 实际应用：表单填充
def fill_form_with_relative_locators(driver):
    # 找到"用户名"标签
    username_label = driver.find_element(
        By.XPATH,
        "//label[contains(text(), '用户名')]"
    )

    # 找到标签右侧的输入框
    username_input = driver.find_element(
        locate_with(By.TAG_NAME, "input").near(username_label)
    )
    username_input.send_keys("testuser")

    # 找到用户名输入框下方的输入框（密码）
    password_input = driver.find_element(
        locate_with(By.TAG_NAME, "input").below(username_input)
    )
    password_input.send_keys("password123")

    # 找到密码输入框下方的按钮
    submit_button = driver.find_element(
        locate_with(By.TAG_NAME, "button").below(password_input)
    )
    submit_button.click()
```

**相对定位方法对比**：

| 方法 | 说明 | 适用场景 |
|------|------|---------|
| `above()` | 垂直方向上方 | 表单中前一个字段 |
| `below()` | 垂直方向下方 | 表单中下一个字段 |
| `to_left_of()` | 水平方向左侧 | 并排按钮 |
| `to_right_of()` | 水平方向右侧 | Label 右侧的输入框 |
| `near()` | 周围 50px 内 | 关联元素（如图标和文字） |

**关键点**：
- 相对定位基于元素的屏幕位置
- 适合 ID/Name 缺失的场景
- 可以组合多个方向提高精确度

## 第 10 题：如何优化元素定位策略以提高测试稳定性？

**答案**：

**1. 定位器优先级策略**：

```python
from selenium import webdriver
from selenium.webdriver.common.by import By

class LocatorStrategy:
    """定位器优先级：ID > Name > CSS > XPath"""

    @staticmethod
    def find_element_robust(driver, locators):
        """按优先级尝试多个定位器"""
        for by, value in locators:
            try:
                element = driver.find_element(by, value)
                if element.is_displayed() and element.is_enabled():
                    return element
            except:
                continue
        raise Exception(f"所有定位器都失败: {locators}")

# 使用示例
username_locators = [
    (By.ID, "username"),
    (By.NAME, "username"),
    (By.CSS_SELECTOR, "input[placeholder='用户名']"),
    (By.XPATH, "//input[@type='text'][1]")
]
username = LocatorStrategy.find_element_robust(driver, username_locators)
```

**2. 使用数据属性（data-* 属性）**：

```python
# 推荐：在 HTML 中添加测试专用属性
# <button data-testid="submit-button">提交</button>

submit_button = driver.find_element(
    By.CSS_SELECTOR,
    "[data-testid='submit-button']"
)

# 或者使用 data-qa 属性
element = driver.find_element(By.CSS_SELECTOR, "[data-qa='login-form']")
```

**3. 避免使用脆弱的定位器**：

```python
# ❌ 不推荐：绝对 XPath
driver.find_element(By.XPATH, "/html/body/div[2]/form/input[1]")

# ✅ 推荐：相对 XPath
driver.find_element(By.XPATH, "//form[@id='login']//input[@name='username']")

# ❌ 不推荐：依赖索引
driver.find_element(By.XPATH, "(//button)[3]")

# ✅ 推荐：使用属性
driver.find_element(By.XPATH, "//button[@type='submit']")

# ❌ 不推荐：依赖文本（可能会变化）
driver.find_element(By.XPATH, "//button[text()='提交']")

# ✅ 推荐：使用 ID 或 class
driver.find_element(By.ID, "submit-btn")
```

**4. 封装页面对象模式（POM）**：

```python
from selenium.webdriver.common.by import By
from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC

class LoginPage:
    # 定位器集中管理
    USERNAME_LOCATORS = [
        (By.ID, "username"),
        (By.NAME, "username"),
        (By.CSS_SELECTOR, "input[type='text']")
    ]

    PASSWORD_LOCATORS = [
        (By.ID, "password"),
        (By.NAME, "password"),
        (By.CSS_SELECTOR, "input[type='password']")
    ]

    SUBMIT_BUTTON = (By.ID, "submit")

    def __init__(self, driver):
        self.driver = driver
        self.wait = WebDriverWait(driver, 10)

    def enter_username(self, username):
        element = self._find_with_fallback(self.USERNAME_LOCATORS)
        element.clear()
        element.send_keys(username)

    def enter_password(self, password):
        element = self._find_with_fallback(self.PASSWORD_LOCATORS)
        element.clear()
        element.send_keys(password)

    def click_submit(self):
        element = self.wait.until(
            EC.element_to_be_clickable(self.SUBMIT_BUTTON)
        )
        element.click()

    def _find_with_fallback(self, locators):
        for by, value in locators:
            try:
                return self.wait.until(
                    EC.presence_of_element_located((by, value))
                )
            except:
                continue
        raise Exception("所有定位器都失败")

# 使用
driver = webdriver.Chrome()
login_page = LoginPage(driver)
login_page.enter_username("test")
login_page.enter_password("password123")
login_page.click_submit()
```

**5. 动态定位器生成**：

```python
class DynamicLocator:
    @staticmethod
    def by_text(tag, text):
        """根据文本内容动态生成 XPath"""
        return (
            By.XPATH,
            f"//{tag}[contains(text(), '{text}')]"
        )

    @staticmethod
    def by_attribute(tag, attr, value):
        """根据属性动态生成 CSS"""
        return (
            By.CSS_SELECTOR,
            f"{tag}[{attr}='{value}']"
        )

    @staticmethod
    def by_partial_id(partial_id):
        """部分 ID 匹配"""
        return (
            By.CSS_SELECTOR,
            f"[id*='{partial_id}']"
        )

# 使用
driver.find_element(*DynamicLocator.by_text("button", "提交"))
driver.find_element(*DynamicLocator.by_attribute("input", "placeholder", "用户名"))
```

**6. 等待元素状态变化**：

```python
from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC

# 等待元素可点击（不仅存在，还要可见且可交互）
element = WebDriverWait(driver, 10).until(

# 等待元素可见
element = WebDriverWait(driver, 10).until(
    EC.visibility_of_element_located((By.ID, "message"))
)

# 等待元素消失
WebDriverWait(driver, 10).until(
    EC.invisibility_of_element_located((By.ID, "loading"))
)
```

**关键点**：
- 优先使用稳定的属性（ID、data-testid）
- 多定位器备选策略提高容错性
- 避免依赖索引和绝对路径
- 页面对象模式集中管理定位器

## 第 11 题：如何处理定位失败的异常？

**答案**：

```python
from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.common.exceptions import (
    NoSuchElementException,
    TimeoutException,
    ElementNotInteractableException,
    StaleElementReferenceException
)
from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC
import logging

driver = webdriver.Chrome()
logging.basicConfig(level=logging.INFO)

# 1. 基本异常捕获
try:
    element = driver.find_element(By.ID, "non-existent")
except NoSuchElementException:
    logging.error("元素未找到")
    # 截图保存
    driver.save_screenshot("element_not_found.png")

# 2. 多异常处理
def safe_find_element(driver, by, value, timeout=10):
    try:
        wait = WebDriverWait(driver, timeout)
        element = wait.until(
            EC.presence_of_element_located((by, value))
        )
        return element
    except TimeoutException:
        logging.error(f"元素定位超时: {by}={value}")
        return None
    except NoSuchElementException:
        logging.error(f"元素不存在: {by}={value}")
        return None
    except Exception as e:
        logging.error(f"未知错误: {e}")
        return None

# 3. 处理过时元素引用（StaleElementReferenceException）
def safe_click(element, max_attempts=3):
    """处理 DOM 刷新导致的元素过时"""
    for attempt in range(max_attempts):
        try:
            element.click()
            return True
        except StaleElementReferenceException:
            if attempt < max_attempts - 1:
                logging.warning(f"元素过时，重试 {attempt + 1}/{max_attempts}")
                # 重新定位元素
                element = driver.find_element(By.ID, element.get_attribute("id"))
            else:
                logging.error("元素过时，达到最大重试次数")
                return False

# 4. 处理元素不可交互
def safe_send_keys(driver, locator, text, timeout=10):
    try:
        wait = WebDriverWait(driver, timeout)
        element = wait.until(EC.element_to_be_clickable(locator))
        element.clear()
        element.send_keys(text)
    except ElementNotInteractableException:
        # 尝试用 JavaScript 输入
        logging.warning("元素不可交互，使用 JavaScript 输入")
        element = driver.find_element(*locator)
        driver.execute_script(
            "arguments[0].value = arguments[1];",
            element,
            text
        )

# 5. 通用重试机制
def retry_on_exception(func, max_attempts=3, delay=1):
    """装饰器：失败时重试"""
    import time

    def wrapper(*args, **kwargs):
        for attempt in range(max_attempts):
            try:
                return func(*args, **kwargs)
            except Exception as e:
                if attempt < max_attempts - 1:
                    logging.warning(f"操作失败，{delay}秒后重试 ({attempt + 1}/{max_attempts})")
                    time.sleep(delay)
                else:
                    logging.error(f"达到最大重试次数，操作失败: {e}")
                    raise
    return wrapper

# 使用装饰器
@retry_on_exception
def find_and_click(driver, by, value):
    element = driver.find_element(by, value)
    element.click()

# 6. 完整的异常处理框架
class ElementHelper:
    def __init__(self, driver, timeout=10):
        self.driver = driver
        self.timeout = timeout
        self.logger = logging.getLogger(__name__)

    def find_element_safe(self, by, value):
        """安全查找元素"""
        try:
            wait = WebDriverWait(self.driver, self.timeout)
            element = wait.until(
                EC.presence_of_element_located((by, value))
            )
            return element
        except TimeoutException:
            self.logger.error(f"元素定位超时: {by}={value}")
            self._take_screenshot(f"timeout_{value}")
            return None
        except NoSuchElementException:
            self.logger.error(f"元素不存在: {by}={value}")
            self._take_screenshot(f"not_found_{value}")
            return None

    def click_safe(self, by, value):
        """安全点击"""
        element = self.find_element_safe(by, value)
        if not element:
            return False

        try:
            # 滚动到元素
            self.driver.execute_script(
                "arguments[0].scrollIntoView(true);",
                element
            )
            # 等待可点击
            wait = WebDriverWait(self.driver, self.timeout)
            wait.until(EC.element_to_be_clickable((by, value)))
            element.click()
            return True
        except ElementNotInteractableException:
            # JavaScript 点击
            self.logger.warning("元素不可交互，使用 JavaScript 点击")
            self.driver.execute_script("arguments[0].click();", element)
            return True
        except Exception as e:
            self.logger.error(f"点击失败: {e}")
            return False

    def _take_screenshot(self, name):
        """失败时截图"""
        filename = f"{name}_{self._get_timestamp()}.png"
        self.driver.save_screenshot(filename)
        self.logger.info(f"截图已保存: {filename}")

    @staticmethod
    def _get_timestamp():
        from datetime import datetime
        return datetime.now().strftime("%Y%m%d_%H%M%S")

# 使用
helper = ElementHelper(driver)
helper.click_safe(By.ID, "submit")
```

**常见异常及处理**：

| 异常 | 原因 | 解决方案 |
|------|------|---------|
| `NoSuchElementException` | 元素不存在 | 检查定位器、等待元素加载 |
| `TimeoutException` | 等待超时 | 增加超时时间、检查元素是否真的会出现 |
| `StaleElementReferenceException` | 元素已从 DOM 移除 | 重新定位元素 |
| `ElementNotInteractableException` | 元素不可交互（被遮挡/隐藏） | 滚动到元素、使用 JavaScript 操作 |
| `ElementClickInterceptedException` | 元素被其他元素遮挡 | 关闭遮挡元素、JavaScript 点击 |

**关键点**：
- 失败时自动截图保存现场
- 使用重试机制处理间歇性问题
- 区分不同异常类型，采取针对性措施

```python
class SVGLocator:
    @staticmethod
    def by_id(svg_id):
        """通过 ID 定位 SVG 元素"""
        return (By.ID, svg_id)

    @staticmethod
    def by_tag_and_attr(tag, attr, value):
        """通过标签名和属性定位"""
        return (
            By.XPATH,
            f"//*[local-name()='{tag}'][@{attr}='{value}']"
        )

    @staticmethod
    def by_tag(tag):
        """定位所有特定标签的 SVG 元素"""
        return (
            By.XPATH,
            f"//*[local-name()='{tag}']"
        )

    @staticmethod
    def by_class(class_name):
        """通过 class 定位"""
        return (
            By.XPATH,
            f"//*[local-name()='svg']//*[contains(@class, '{class_name}')]"
        )

# 使用
driver.find_element(*SVGLocator.by_id("circle1"))
driver.find_element(*SVGLocator.by_tag_and_attr("circle", "cx", "50"))
```

**关键点**：
- SVG 元素可以用 ID 或 CSS Selector 定位（简单场景）
- 复杂场景建议用 XPath 的 `local-name()`
- SVG 元素可以像普通元素一样点击和悬停

```python
class LoginPageLocators:
    """登录页面定位器（多层备选）"""

    # 用户名输入框（按优先级排列）
    USERNAME_PRIMARY = (By.ID, "username")
    USERNAME_SECONDARY = (By.NAME, "username")
    USERNAME_TERTIARY = (By.CSS_SELECTOR, "input[type='text'][placeholder*='用户名']")
    USERNAME_FALLBACK = (By.XPATH, "//form[@id='login-form']//input[@type='text'][1]")

    # 密码输入框
    PASSWORD_PRIMARY = (By.ID, "password")
    PASSWORD_SECONDARY = (By.NAME, "password")
    PASSWORD_TERTIARY = (By.CSS_SELECTOR, "input[type='password']")

    # 提交按钮
    SUBMIT_PRIMARY = (By.ID, "login-submit")
    SUBMIT_SECONDARY = (By.CSS_SELECTOR, "button[type='submit']")
    SUBMIT_TERTIARY = (By.XPATH, "//button[contains(text(), '登录')]")

    @classmethod
    def get_username_locators(cls):
        """获取用户名输入框的所有定位器"""
        return [
            cls.USERNAME_PRIMARY,
            cls.USERNAME_SECONDARY,
            cls.USERNAME_TERTIARY,
            cls.USERNAME_FALLBACK
        ]

# page.py - 页面对象
class LoginPage:
    def __init__(self, driver):
        self.driver = driver

    def enter_username(self, username):
        element = self._find_with_fallback(
            LoginPageLocators.get_username_locators()
        )
        element.send_keys(username)

    def _find_with_fallback(self, locators):
        for by, value in locators:
            try:
                element = self.driver.find_element(by, value)
                if element.is_displayed():
                    return element
            except:
                continue
        raise Exception("所有定位器都失败")
```

**4. 动态定位器适配**：

```python
class AdaptiveLocator:
    """自适应定位器（运行时检测页面结构）"""

    def __init__(self, driver):
        self.driver = driver
        self._cache = {}

    def find_login_button(self):
        """智能查找登录按钮"""
        cache_key = "login_button"

        # 尝试从缓存获取
        if cache_key in self._cache:
            try:
                return self.driver.find_element(*self._cache[cache_key])
            except:
                del self._cache[cache_key]

        # 尝试多种定位策略
        strategies = [
            (By.ID, "login"),
            (By.ID, "submit"),
            (By.CSS_SELECTOR, "button[type='submit']"),
            (By.XPATH, "//button[contains(text(), '登录')]"),
            (By.XPATH, "//button[contains(text(), 'Login')]"),
            (By.XPATH, "//button[contains(text(), 'Sign In')]"),
            (By.CSS_SELECTOR, ".btn-login"),
            (By.CSS_SELECTOR, ".login-button"),
        ]

        for by, value in strategies:
            try:
                element = self.driver.find_element(by, value)
                if element.is_displayed() and element.is_enabled():
                    # 缓存成功的定位器
                    self._cache[cache_key] = (by, value)
                    return element
            except:
                continue

        raise Exception("无法找到登录按钮")

# 使用
locator = AdaptiveLocator(driver)
login_button = locator.find_login_button()
```

**6. 定位器自动修复**：

```python
class SelfHealingLocator:
    """自修复定位器（失败时尝试相似元素）"""

    def __init__(self, driver):
        self.driver = driver

    def find_similar_element(self, failed_locator):
        """当原定位器失败时，尝试找相似元素"""
        by, value = failed_locator

        if by == By.ID:
            # ID 失败，尝试部分匹配
            return self.driver.find_element(
                By.CSS_SELECTOR,
                f"[id*='{value}']"
            )
        elif by == By.CSS_SELECTOR:
            # CSS 失败，尝试 XPath
            return self.driver.find_element(
                By.XPATH,
                self._css_to_xpath(value)
            )

        raise Exception("无法找到相似元素")

    def _css_to_xpath(self, css):
        """简单的 CSS 到 XPath 转换"""
        # 这是简化版本，实际项目可用 cssselect 库
        if css.startswith("#"):
            element_id = css[1:]
            return f"//*[@id='{element_id}']"
        elif css.startswith("."):
            class_name = css[1:]
            return f"//*[contains(@class, '{class_name}')]"
        return f"//{css}"
```

**关键点**：
- 多层备选定位器提高容错性
- 使用测试专用属性（data-test-id）隔离变化
- 定位器集中管理，便于维护
- 考虑使用自适应和自修复机制

## 第 15 题：如何使用 JavaScript 辅助元素定位？

**答案**：

```python
from selenium import webdriver
from selenium.webdriver.common.by import By

driver = webdriver.Chrome()
driver.get("https://www.example.com")

# 1. 通过 JavaScript 查找元素
element = driver.execute_script("""
    return document.getElementById('username');
""")

# 2. 使用 querySelector（CSS Selector）
element = driver.execute_script("""
    return document.querySelector('#username');
""")

# 3. 使用 querySelectorAll（返回多个元素）
elements = driver.execute_script("""
    return document.querySelectorAll('.item');
""")

# 4. 通过文本内容查找
element = driver.execute_script("""
    var elements = Array.from(document.querySelectorAll('button'));
    return elements.find(el => el.textContent.includes('提交'));
""")

# 5. 查找包含特定属性的元素
element = driver.execute_script("""
    return document.querySelector('[data-action="submit"]');
""")

# 6. 深度查找（包括 Shadow DOM）
element = driver.execute_script("""
    function deepFind(root, selector) {
        // 在当前层级查找
        let element = root.querySelector(selector);
        if (element) return element;

        // 递归查找 Shadow DOM
        let shadowHosts = root.querySelectorAll('*');
        for (let host of shadowHosts) {
            if (host.shadowRoot) {
                element = deepFind(host.shadowRoot, selector);
                if (element) return element;
            }
        }
        return null;
    }
    return deepFind(document, arguments[0]);
""", "#shadow-element")

# 7. 获取所有可见元素
visible_elements = driver.execute_script("""
    return Array.from(document.querySelectorAll('*')).filter(el => {
        var style = window.getComputedStyle(el);
        return style.display !== 'none' &&
               style.visibility !== 'hidden' &&
               el.offsetParent !== null;
    });
""")

# 8. 查找视口内的元素
elements_in_viewport = driver.execute_script("""
    return Array.from(document.querySelectorAll('*')).filter(el => {
        var rect = el.getBoundingClientRect();
        return rect.top >= 0 &&
               rect.left >= 0 &&
               rect.bottom <= window.innerHeight &&
               rect.right <= window.innerWidth;
    });
""")

# 9. 通过自定义条件查找
element = driver.execute_script("""
    return Array.from(document.querySelectorAll('input')).find(el => {
        return el.type === 'text' &&
               el.placeholder.includes('用户名') &&
               !el.disabled;
    });
""")

# 10. 查找最近的可点击祖先元素
element = driver.find_element(By.ID, "inner-span")
clickable_parent = driver.execute_script("""
    var el = arguments[0];
    while (el && el !== document.body) {
        if (el.onclick || el.tagName === 'A' || el.tagName === 'BUTTON') {
            return el;
        }
        el = el.parentElement;
    }
    return null;
""", element)

# 11. 封装 JavaScript 定位器辅助类
class JSLocator:
    def __init__(self, driver):
        self.driver = driver

    def find_by_text(self, tag, text):
        """通过文本内容查找"""
        return self.driver.execute_script("""
            var elements = Array.from(document.querySelectorAll(arguments[0]));
            return elements.find(el => el.textContent.trim() === arguments[1]);
        """, tag, text)

    def find_by_partial_text(self, tag, partial_text):
        """通过部分文本查找"""
        return self.driver.execute_script("""
            var elements = Array.from(document.querySelectorAll(arguments[0]));
            return elements.find(el => el.textContent.includes(arguments[1]));
        """, tag, partial_text)

    def find_visible_elements(self, selector):
        """查找所有可见元素"""
        return self.driver.execute_script("""
            return Array.from(document.querySelectorAll(arguments[0])).filter(el => {
                var style = window.getComputedStyle(el);
                return style.display !== 'none' &&
                       style.visibility !== 'hidden' &&
                       el.offsetWidth > 0 &&
                       el.offsetHeight > 0;
            });
        """, selector)

    def find_by_attribute_contains(self, attr, value):
        """查找属性包含特定值的元素"""
        return self.driver.execute_script(f"""
            return document.querySelector('[{attr}*="{value}"]');
        """)

# 使用
js_locator = JSLocator(driver)
submit_button = js_locator.find_by_text("button", "提交")
visible_inputs = js_locator.find_visible_elements("input")
```

**JavaScript 定位的优势**：

| 场景 | 原因 |
|------|------|
| **复杂条件筛选** | JavaScript 可以使用 `filter()`, `find()` 等方法 |
| **Shadow DOM** | 可以递归访问 Shadow Root |
| **动态计算** | 可以基于样式、位置等动态条件查找 |
| **性能** | 某些情况下比 XPath 更快 |
| **绕过限制** | 可以操作不可见或禁用的元素 |

**关键点**：
- JavaScript 定位适合复杂条件
- 返回的元素可以直接用于 Selenium 操作
- 注意跨浏览器兼容性
