## 🎯 精简版特点

- **60 题精华**：只保留最核心、最高频的面试问题
- **主题整合**：将相关知识点合并，更便于系统学习

- **实战导向**：聚焦实际工作场景和最佳实践

## 📖 Selenium 面试题（30 题）

### 主题分类
1. **基础和定位**（10 题）
   - Selenium 架构、WebDriver 原理
   - 8 种定位方法、CSS/XPath
   - 动态元素、Shadow DOM、相对定位

2. **等待和交互**（10 题）
   - 三种等待机制、Expected Conditions
   - ActionChains、键盘鼠标操作
   - 文件上传下载、iframe/窗口切换

3. **高级应用**（10 题）
   - JavaScript 执行、POM 模式
   - 数据驱动、Grid 并发、CDP
   - 反爬虫、性能优化、实战问题

## 🚀 Cypress 面试题（30 题）

### 主题分类
1. **基础和选择器**（10 题）
   - Cypress 特点、与 Selenium 对比
   - 选择器策略、DOM 遍历
   - 基本命令（visit、type、click）

2. **异步和断言**（10 题）
   - 异步处理、命令队列、自动等待
   - should/expect 断言
   - 测试组织、钩子函数

3. **网络和实践**（10 题）
   - cy.intercept 网络拦截
   - 配置文件、自定义命令
   - CI/CD 集成、最佳实践
