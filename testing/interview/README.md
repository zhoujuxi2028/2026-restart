# 测试自动化框架面试题集

本目录包含主流测试自动化框架的面试题，帮助准备测试开发和自动化测试工程师岗位。

## 📚 目录结构

```
interview/
├── README.md          # 本文件
├── Selenium/          # Selenium 面试题（90 题）
└── Cypress/           # Cypress 面试题（90 题）
```

## 🎯 学习目标

- **Selenium**: 掌握 WebDriver、元素定位、等待机制、框架设计等核心概念
- **Cypress**: 理解现代前端测试框架的特点、异步处理、网络拦截等高级功能
- **对比思维**: 理解两种框架的优劣势，能够根据项目需求选择合适的工具

## 📖 Selenium 面试题（90 题）

### 难度分布
- 基础题（1-2 年经验）：45 题
- 进阶题（3-4 年经验）：35 题
- 高级题（5+ 年经验）：10 题

### 主题分类
1. **基础概念**（15 题）- Selenium 生态、WebDriver 原理、环境配置
2. **元素定位**（15 题）- 8 种定位方式、CSS 选择器、XPath 实战
3. **等待机制**（12 题）- 隐式等待、显式等待、自定义等待条件
4. **操作交互**（15 题）- 表单操作、窗口切换、鼠标键盘事件
5. **高级特性**（10 题）- JavaScript 执行、截图、文件处理
6. **框架设计**（12 题）- POM 模式、数据驱动、框架架构
7. **并发执行**（6 题）- Selenium Grid、并行测试
8. **实战问题**（5 题）- 调试技巧、常见坑点

## 🚀 Cypress 面试题（90 题）

### 难度分布
- 基础题（1-2 年经验）：50 题
- 进阶题（3-4 年经验）：30 题
- 高级题（5+ 年经验）：10 题

### 主题分类
1. **基础概念**（15 题）- Cypress 特点、架构、与 Selenium 对比
2. **选择器和命令**（20 题）- 查询命令、操作命令、链式调用
3. **异步处理**（15 题）- Promise 机制、retry-ability
4. **断言和测试**（15 题）- should/expect、测试组织
5. **网络请求**（12 题）- cy.intercept、Mock 数据
6. **配置和插件**（8 题）- cypress.config.js、自定义命令
7. **CI/CD 集成**（5 题）- Docker、持续集成

## 📅 学习计划建议

### 快速准备（1 周）
- **Day 1-2**: Selenium 基础概念 + 元素定位
- **Day 3-4**: Selenium 等待机制 + 操作交互
- **Day 5**: Cypress 基础概念 + 选择器命令
- **Day 6**: Cypress 异步处理 + 网络请求
- **Day 7**: 两个框架的对比和实战问题复习

### 系统准备（2 周）
- **Week 1**: 完整学习 Selenium 所有主题（每天 2-3 小时）
- **Week 2**: 完整学习 Cypress 所有主题 + 实战练习

## 🎓 使用建议

1. **循序渐进**: 按照文件顺序学习，先打好基础再学习高级内容
2. **动手实践**: 每个知识点都要实际编码验证，不要只看不练
3. **对比学习**: 学习过程中思考两个框架的异同点
4. **总结归纳**: 每个主题学完后总结核心要点
