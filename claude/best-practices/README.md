## 💡 核心原则

### ✅ 最佳实践的特征
- **可操作**：有具体步骤，能立即应用
- **有效果**：经过验证，确实有用
- **有案例**：配有实际例子
- **可重复**：别人也能照做并成功

### ❌ 避免的误区
- 过于理论化，缺少实践指导
- 堆砌技巧，没有优先级
- 只说"应该"，不说"如何"
- 忽略具体场景，泛泛而谈
