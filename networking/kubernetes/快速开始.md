# Kubernetes 快速开始指南

## 🚀 5分钟快速部署

### 步骤1：启用 Docker Desktop Kubernetes

1. 打开 **Docker Desktop**
2. 点击右上角 **⚙️ 设置图标**
3. 左侧菜单选择 **Kubernetes**
4. 勾选 ☑️ **Enable Kubernetes**
5. 点击 **Apply & Restart**
6. 等待 Kubernetes 图标变为 🟢 绿色（约2-3分钟）

### 步骤2：验证安装

```bash
# 查看集群信息
kubectl cluster-info

# 应该看到类似输出：
# Kubernetes control plane is running at https://kubernetes.docker.internal:6443
# CoreDNS is running at https://kubernetes.docker.internal:6443/api/v1/namespaces/kube-system/services/kube-dns:dns/proxy

# 查看节点
kubectl get nodes

# 应该看到：
# NAME             STATUS   ROLES           AGE   VERSION
# docker-desktop   Ready    control-plane   5m    v1.x.x

# 查看系统组件
kubectl get pods -n kube-system
```

如果以上命令都正常工作，恭喜！您的 K8S 环境已经准备就绪！

### 步骤3：部署第一个应用

```bash
# 进入项目目录
cd /Users/michaelzhou/Documents/github/2026-restart/networking/kubernetes

# 创建一个 nginx 应用
kubectl create deployment nginx --image=nginx:alpine

# 查看部署
kubectl get deployments
kubectl get pods

# 暴露服务
kubectl expose deployment nginx --port=80 --type=NodePort

# 查看服务
kubectl get services

# 访问服务（端口转发到本地）
kubectl port-forward service/nginx 8080:80

# 打开浏览器访问 http://localhost:8080
# 应该能看到 nginx 欢迎页面
```

### 步骤4：清理测试资源

```bash
# 删除刚才创建的资源
kubectl delete service nginx
kubectl delete deployment nginx

# 验证清理
kubectl get all
```

## 🛠️ 常用命令速查卡

### 查看资源
```bash
kubectl get nodes                    # 节点
kubectl get pods                     # Pod（当前命名空间）
kubectl get pods -A                  # Pod（所有命名空间）
kubectl get deployments              # 部署
kubectl get services                 # 服务
kubectl get all                      # 所有资源
```

### 详细信息
```bash
kubectl describe pod <name>          # Pod 详情
kubectl logs <pod-name>             # 查看日志
kubectl logs -f <pod-name>          # 实时日志
kubectl exec -it <pod> -- bash      # 进入容器
```

### 创建/更新/删除
```bash
kubectl apply -f <file.yaml>        # 应用配置
kubectl delete -f <file.yaml>       # 删除配置
kubectl delete pod <name>           # 删除 Pod
kubectl scale deployment <name> --replicas=3  # 扩缩容
```

### 调试
```bash
kubectl get events                   # 查看事件
kubectl describe <resource> <name>   # 详细描述
kubectl port-forward <pod> 8080:80  # 端口转发
```
