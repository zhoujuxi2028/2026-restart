## 1. Cypress 是什么？它的核心优势是什么？

**答案：**
Cypress 是一个为现代 Web 应用构建的前端测试框架，直接运行在浏览器中。

**核心优势：**
- **直接在浏览器中运行**：与应用代码运行在同一事件循环中
- **自动等待**：内置智能等待机制，无需显式添加 wait
- **时间旅行**：可以查看每个命令执行时的快照
- **实时重载**：测试代码修改后自动重新运行
- **网络流量控制**：轻松 stub 和 mock 网络请求

**代码示例：**
```javascript
// Cypress 的简洁语法
describe('登录功能', () => {
  it('应该成功登录', () => {
    cy.visit('/login')
    cy.get('[data-cy="username"]').type('testuser')
    cy.get('[data-cy="password"]').type('password123')
    cy.get('[data-cy="login-btn"]').click()
    cy.url().should('include', '/dashboard')
  })
})
```

## 2. Cypress 与 Selenium 有什么主要区别？

**答案：**
| 特性 | Cypress | Selenium |
|------|---------|----------|
| 架构 | 直接在浏览器中运行 | 通过 WebDriver 协议远程控制 |
| 自动等待 | 内置智能等待 | 需要显式等待 |
| 速度 | 更快（同一运行循环） | 较慢（网络通信开销） |
| 浏览器支持 | Chrome、Edge、Firefox、Electron | 所有主流浏览器 |
| 编程语言 | 仅 JavaScript | 多种语言 |
| 网络控制 | 原生支持拦截和 Mock | 需要额外工具 |
| 调试体验 | 优秀（时间旅行、快照） | 一般 |

**代码对比：**
```javascript
// Cypress - 简洁且自动等待
cy.get('#submit-btn').click()

// Selenium - 需要显式等待
const element = await driver.wait(
  until.elementLocated(By.id('submit-btn')),
  5000
)
await element.click()
```

## 3. Cypress 如何选择 DOM 元素？推荐的选择器策略是什么？

**答案：**
Cypress 使用 jQuery 风格的选择器，但推荐使用 `data-*` 属性。

**选择器优先级（从高到低）：**
1. **data-cy / data-test / data-testid**：最佳实践
2. **id 属性**：唯一且稳定
3. **class 属性**：注意样式变更的影响
4. **标签名**：不推荐，容易变化

**代码示例：**
```javascript
// ✅ 最佳实践 - 使用 data-cy
cy.get('[data-cy="submit-button"]').click()

// ✅ 好的实践 - 使用 id
cy.get('#login-form').submit()

// ⚠️ 可接受 - 使用 class（注意样式变更）
cy.get('.btn-primary').click()

// ❌ 不推荐 - 使用文本内容（国际化会导致失败）
cy.contains('Submit').click()

// ✅ 组合选择器
cy.get('[data-cy="user-list"]')
  .find('[data-cy="user-item"]')
  .first()
```

## 4. `cy.get()` 和 `cy.contains()` 有什么区别？

**答案：**
- `cy.get()`：通过选择器查找 DOM 元素
- `cy.contains()`：通过文本内容查找元素

**代码示例：**
```javascript
// cy.get() - 选择器匹配
cy.get('[data-cy="title"]')  // 查找具有 data-cy="title" 的元素

// cy.contains() - 文本内容匹配
cy.contains('欢迎回来')  // 查找包含"欢迎回来"文本的元素

// cy.contains() 可以组合选择器和文本
cy.contains('button', '提交')  // 查找包含"提交"文本的 button 元素

// 实际应用场景
cy.get('[data-cy="nav-menu"]')
  .contains('用户中心')
  .click()

// 验证特定元素包含特定文本
cy.get('[data-cy="error-message"]')
  .should('contain', '用户名不能为空')
```

## 5. 如何在 Cypress 中进行 DOM 遍历？

**答案：**
Cypress 提供了丰富的 DOM 遍历命令，类似 jQuery。

**常用遍历方法：**
```javascript
// 父子关系
cy.get('[data-cy="parent"]')
  .children()          // 获取直接子元素
  .find('.child')      // 查找后代元素
  .parent()            // 获取父元素
  .parents()           // 获取所有祖先元素

// 兄弟关系
cy.get('[data-cy="item"]')
  .next()              // 下一个兄弟元素
  .prev()              // 上一个兄弟元素
  .siblings()          // 所有兄弟元素

// 筛选
cy.get('.list-item')
  .first()             // 第一个元素
  .last()              // 最后一个元素
  .eq(2)               // 索引为 2 的元素
  .filter('.active')   // 过滤元素

// 实际应用：导航到表格特定单元格
cy.get('[data-cy="user-table"]')
  .find('tbody tr')
  .eq(1)
  .find('td')
  .eq(2)
  .should('contain', 'admin@example.com')
```

## 6. `cy.visit()` 的作用是什么？如何配置基础 URL？

**答案：**
`cy.visit()` 用于访问指定 URL，可以在 `cypress.config.js` 中配置 `baseUrl`。

**配置文件：**
```javascript
// cypress.config.js
const { defineConfig } = require('cypress')

module.exports = defineConfig({
  e2e: {
    baseUrl: 'http://localhost:3000',
    setupNodeEvents(on, config) {
      // 事件监听器
    },
  },
})
```

**使用示例：**
```javascript
// 有 baseUrl 配置时
cy.visit('/')              // 访问 http://localhost:3000/
cy.visit('/dashboard')     // 访问 http://localhost:3000/dashboard

// 完整 URL（会忽略 baseUrl）
cy.visit('https://example.com')

// 带选项
cy.visit('/login', {
  timeout: 10000,          // 超时时间
  onBeforeLoad: (win) => { // 页面加载前执行
    win.localStorage.setItem('token', 'abc123')
  },
  onLoad: (win) => {       // 页面加载后执行
    console.log('页面已加载')
  }
})
```

## 7. 如何使用 `cy.type()` 输入文本？有哪些常用选项？

**答案：**
`cy.type()` 用于在输入框中输入文本，支持特殊键和选项。

**代码示例：**
```javascript
// 基本输入
cy.get('[data-cy="username"]').type('testuser')

// 特殊键
cy.get('[data-cy="search"]')
  .type('Cypress{enter}')  // 输入后按回车

// 常用特殊键
cy.get('input')
  .type('{backspace}')     // 退格
  .type('{del}')           // 删除
  .type('{enter}')         // 回车
  .type('{esc}')           // ESC
  .type('{selectall}')     // 全选
  .type('{ctrl+c}')        // 复制
  .type('{leftarrow}')     // 左箭头

// 选项
cy.get('[data-cy="password"]').type('secret123', {
  delay: 100,              // 每个字符输入延迟（毫秒）
  force: true,             // 强制输入（即使元素被覆盖）
  sensitive: true,         // 敏感数据（不记录到日志）
})

// 清空后输入
cy.get('[data-cy="email"]')
  .clear()
  .type('newemail@example.com')
```

## 8. 如何点击元素？`cy.click()` 有哪些常用选项？

**答案：**
`cy.click()` 用于点击元素，支持多种点击方式和位置选项。

**代码示例：**
```javascript
// 基本点击
cy.get('[data-cy="submit"]').click()

// 点击特定位置
cy.get('.canvas').click(100, 200)  // 点击坐标 (100, 200)
cy.get('.button').click('topLeft') // 点击左上角
// 位置选项：center, topLeft, topRight, bottomLeft, bottomRight

// 多次点击
cy.get('[data-cy="counter"]').dblclick()  // 双击
cy.get('[data-cy="menu"]').rightclick()   // 右键点击

// 强制点击（忽略可见性检查）
cy.get('[data-cy="hidden-btn"]').click({ force: true })

// 多个元素点击
cy.get('[data-cy="checkbox"]').click({ multiple: true })

// 实际场景：点击被遮挡的元素
cy.get('[data-cy="modal-close"]').click({ force: true })

// 链式操作
cy.get('[data-cy="item"]')
  .first()
  .click()
  .should('have.class', 'active')
```

## 9. `cy.get()` 和 `cy.find()` 有什么区别？

**答案：**
- `cy.get()`：从整个 DOM 中查找元素（可以作为起始命令）
- `cy.find()`：从前一个命令的结果中查找子元素（必须链式调用）

**代码示例：**
```javascript
// cy.get() - 全局查找
cy.get('[data-cy="user-list"]')  // ✅ 可以独立使用

// cy.find() - 必须链式调用
cy.find('[data-cy="item"]')      // ❌ 错误：不能独立使用

// 正确使用 cy.find()
cy.get('[data-cy="user-list"]')
  .find('[data-cy="user-item"]')  // ✅ 在 user-list 内查找

// 实际应用：在特定容器内查找元素
cy.get('[data-cy="sidebar"]')
  .find('[data-cy="nav-link"]')
  .should('have.length', 5)

// 组合使用
cy.get('[data-cy="table"]')
  .find('tbody tr')              // 查找表格行
  .eq(2)                         // 第 3 行
  .find('td')                    // 查找单元格
  .first()                       // 第一个单元格
  .should('contain', 'John')
```

## 10. 如何使用 `cy.within()` 限定查找范围？

**答案：**
`cy.within()` 用于限定后续命令的查找范围，适合在特定容器内操作。

**代码示例：**
```javascript
// 基本使用
cy.get('[data-cy="login-form"]').within(() => {
  cy.get('[data-cy="username"]').type('testuser')
  cy.get('[data-cy="password"]').type('pass123')
  cy.get('[data-cy="submit"]').click()
})

// 表单场景
cy.get('[data-cy="user-form"]').within(() => {
  cy.get('input[name="firstName"]').type('张')
  cy.get('input[name="lastName"]').type('三')
  cy.get('select[name="gender"]').select('male')
  cy.get('button[type="submit"]').click()
})

// 表格行操作
cy.get('[data-cy="user-table"]')
  .find('tbody tr')
  .eq(1)
  .within(() => {
    cy.get('td').eq(0).should('contain', 'John')
    cy.get('td').eq(1).should('contain', 'Admin')
    cy.get('[data-cy="edit-btn"]').click()
  })

// 嵌套容器
cy.get('[data-cy="sidebar"]').within(() => {
  cy.get('[data-cy="menu"]').within(() => {
    cy.get('[data-cy="profile-link"]').click()
  })
})

// 实际应用：多个相似表单
cy.get('[data-cy="shipping-form"]').within(() => {
  cy.get('[name="address"]').type('123 Main St')
})
cy.get('[data-cy="billing-form"]').within(() => {
  cy.get('[name="address"]').type('456 Oak Ave')
})
```
