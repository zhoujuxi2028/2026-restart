## 🎯 练习内容（8个练习）

1. **基础日志中间件**（基础） - 记录请求信息
2. **CORS 配置**（基础） - 跨域资源共享
3. **请求 ID 中间件**（中等） - 请求追踪
4. **认证中间件**（中等） - API 密钥验证
5. **性能监控中间件**（困难） - 响应时间统计
6. **速率限制中间件**（困难） - 防止滥用
7. **错误处理中间件**（挑战） - 统一错误处理
8. **完整中间件栈**（综合） - 企业级配置

## 🎯 学习重点
- ✅ 理解中间件的洋葱模型
- ✅ 掌握 CORS 配置的各种场景
- ✅ 学会编写自定义中间件
- ✅ 理解中间件的执行顺序

## 💡 关键概念
- **中间件**: 请求和响应的拦截器
- **CORS**: 跨域资源共享配置
- **执行顺序**: 洋葱模型（先进后出）
- **状态管理**: request.state 的使用
- **性能监控**: 响应时间统计
